# USS精算書 自動取込 セットアップ手順（`USS精算.gs`）

USSから届く**パスワード付き精算書メール**を自動で開き、加盟店シートへ転写し、
**本部手数料（落札11,000＋振込550＝11,550円）**を加算、さらに**入金消し込み**まで行います。

---

## 全体の流れ

```
USS精算書メール（添付PDF・パスワード付き）
   └→ Gmailが自動検知（差出人/件名で絞り込み）
        └→ PDF.coで自動復号（Config.gsのUSS_PDF_PASSWORDを自動入力）
             └→ 内容を「USS精算_加盟店」シートへ転写
                  └→ 本部手数料 11,550円を自動加算 → 加盟店請求額を算出
                       └→ 銀行CSVと突合して「入金消し込み」
```

> **重要**：GAS単体では暗号PDFを開けません。パスワードが分かっていても復号処理が別途必要なため、
> `PDF.co`（方式A）に復号を委譲します。社外にファイルを出したくない場合は、
> **USS管理画面からCSVをダウンロード**して `USS_DECRYPT_MODE:"none"` で運用してください（PDF.co不要）。

---

## 1. 事前準備

| 項目 | 内容 |
|---|---|
| スプレッドシート | 既存の管理DBを流用（`Config.gs` の `SPREADSHEET_ID`） |
| PDF.co アカウント | https://pdf.co でサインアップ → APIキー取得（方式Aのみ必要） |
| パスワード | USSから事前共有される固定パスワード（例: `U3472`） |

## 2. Config.gs の設定（⑦⑧）

```js
USS_MAIL_FROM:    "uss",        // 精算書メールの差出人（部分一致）
USS_MAIL_SUBJECT: "精算",       // 件名キーワード
USS_PDF_PASSWORD: "U3472",      // 添付パスワード（実運用の値に）
USS_DECRYPT_MODE: "pdfco",      // "pdfco"=PDF復号 / "none"=CSV運用
PDFCO_API_KEY:    "xxxxxxxx",   // PDF.coのAPIキー（pdfco時のみ）
USS_FEE_RAKUSATSU: 11000,       // 本部手数料：落札
USS_FEE_FURIKOMI:  550,         // 本部手数料：振込
USS_BANK_CSV_FOLDER_ID: "",     // 銀行入金CSVを置くフォルダID（空欄可）
```

## 3. 初回実行（GASエディタの関数選択 → 実行）

1. `setupUss()` … シート「USS精算_加盟店」「入金消し込み」を作成
2. `installUssTriggers()` … 取込15分毎・消し込み毎日9時のトリガー設置
   - 初回は権限承認（Gmail/Drive/外部通信）を求められます → 許可

## 4. 動作確認

| 関数 | 確認内容 |
|---|---|
| `testFee()` | 本部手数料合計＝11,550円 になるか |
| `testParseUssText()` | サンプル文字列から明細を抽出できるか |
| `runUssSettlement()` | 手動実行で実メールを取込（ログ確認） |
| `runReconciliation()` | 入金CSVを読み込み消し込み |

---

## 入金消し込みの使い方

1. ネットバンキングで**入出金明細CSV**をダウンロード
2. Driveの `USS入金CSV` フォルダ（または `USS_BANK_CSV_FOLDER_ID` 指定先）に置く
3. `runReconciliation()` が「未入金」の加盟店請求額と**金額一致**で自動マッチ → 消込済に更新
   - 振込手数料の差などは `USS_MATCH_TOLERANCE`（円）で吸収

---

## USSの精算書フォーマットに合わせた調整（重要）

`USS精算.gs` の **`parseUssText_()` / `parseUssCsv_()`** は汎用の抽出ロジックです。
実際の精算書サンプルを1枚いただければ、**出品番号・車名・落札価格などの抽出精度を上げる**
正規表現に調整します（ここが唯一、実物合わせが必要な箇所です）。

| 添付形式 | 対応 | 調整対象 |
|---|---|---|
| パスワード付きPDF | `pdfco`で復号 → `parseUssText_()` | テキストの抽出パターン |
| CSV（USS画面DL） | 直接読込 → `parseUssCsv_()` | ヘッダ名の対応表 |

## 費用の目安（方式A）

- PDF.co：従量課金。精算書1通あたり数円程度（月数百円〜）。
- CSV運用（方式C）なら外部API不要＝無料。
