# BUYMO ステップメール＆会員誘導 セットアップ

外部ツール不要・WordPress不要。**GAS（スプレッドシート＋Gmail）だけ**で、査定申込者へ自動フォローメールを送り、会員専用ページへ誘導します。

## 仕組み（全体像）

```
BUYMO 査定/問い合わせフォーム（buymo.js, type:"buymo"）
        │  POST
        ▼
GAS WebApp.gs  handleBuymoLead_()
        ├─ 問い合わせシートに記録 ＋ LINE/Slack通知
        └─ StepMail.gs enrollStepMail_() … 「BUYMOステップメール」シートに登録
        ▼
GAS トリガー（時間主導）runStepMails()  … Day0/1/3/7 のメールを自動送信
        ▼   各メール内の「会員ページCTA」
会員専用ページ（login.html → mypage.html）へ誘導
```

サイト側でも、**フォーム送信完了メッセージ**と**LP下部の会員誘導バナー**から会員ページへ誘導します。

## 配信内容（初期設定）

| タイミング | 件名 | 主な内容 |
|---|---|---|
| Day 0（即時） | 査定のお申し込みありがとうございます | 受付お礼＋会員ページ案内 |
| Day 1 | 買取の流れと、よくあるご質問 | 安心材料＋会員ページ |
| Day 3 | 今が売りどき？査定は無料です | 売却促進＋会員ページ |
| Day 7 | まだ間に合います／無料査定のご案内 | 再案内＋電話導線 |

文言・通数は `gas/StepMail.gs` の `stepSequence_()` を編集すれば自由に変更できます。

## セットアップ手順

1. **GASに `StepMail.gs` を追加**（このリポジトリの `gas/StepMail.gs` を貼り付け）。`WebApp.gs` も最新（`type:"buymo"` 対応版）に更新。
2. **`stepCfg_()` を編集**（`gas/StepMail.gs` 冒頭）
   - `SITE_URL`：公開ドメイン（例 `https://buymo.example.jp`）
   - `MEMBER_URL`：会員ページ（既定 `/login.html`）
   - `TEL`：表示電話番号
3. **トリガー登録**：GASエディタ左「⏰ トリガー」→「トリガーを追加」
   - 関数：`runStepMails`
   - イベントのソース：**時間主導型** → 日タイマー（例：午前9〜10時）。こまめに送るなら時タイマー（1時間おき）。
4. **初回承認**：`runStepMails` を手動実行 → Gmail送信の権限を承認。
5. **フォーム接続**：`site/assets/js/buymo.js` の `ENDPOINT` にウェブアプリURL（`…/exec`）を設定（未設定だとデモ動作で送信されません）。

## 注意・運用メモ

- 送信は**実行ユーザーのGmail**から行われます。無料Googleアカウントは1日あたり約100通、Workspaceは約1,500通の上限があります（大量配信時は外部配信ツールへ移行を検討）。
- **配信停止**：各メール末尾の「配信を停止する」リンク（`?action=unsub&t=…`）で、その人を「停止」に変更します（`WebApp.gs` の doGet が処理）。
- **重複防止**：同じメールアドレスが「配信中」の間は二重登録しません。
- 会員ページ（`login.html`/`mypage.html`）は現状AUC-AGENTの会員サイトを流用。BUYMO専用のマイページにする場合は別途デザイン差し替えが可能です。
- 法令：特定電子メール法に基づき、**送信者情報の明示**と**オプトアウト（配信停止）導線**を必ず維持してください（本実装は両方を本文に含めています）。本格運用前に、申込フォームに「メール配信への同意」明記を推奨。
