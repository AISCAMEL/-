# LPtools で AUC-AGENT のLPを作るためのガイド

## まず結論

| やりたいこと | LPtoolsで可能？ | 方法 |
|---|---|---|
| LP本体のデザイン（ヒーロー/流れ/料金/FAQ/お客様の声/CTA） | ✅ 可能 | LPtoolsの各ブロックで組む（下の対応表） |
| サイト上の**料金シミュレーション** | ✅ 可能（埋め込み） | 「カスタムHTML」ブロックに `simulator-embed.html` を貼る |
| **会員サイト・マイページ・自動オーダー** | ⚠️ LPtools単体では不可 | WordPressの会員制プラグイン、または別途用意した会員ページ＋GAS自動化で対応 |

> **前提**：LPtools は **WordPress用の有料プラグイン**（Catch the Web社）です。利用にはWordPressサイトとライセンスが必要です。LPtoolsは「集客用のLP（静的ページ）」を作るツールで、会員ログインや注文管理などの動的機能は守備範囲外です。そこは既に作った会員サイト＋GAS自動化で補完する構成になります。

---

## 1. シミュレーターの埋め込み手順（いちばん重要）

1. LPtoolsの編集画面で、料金セクションを置きたい場所に **「カスタムHTML」ブロック**（WordPress標準）を追加
2. `site/embed/simulator-embed.html` の中身を**丸ごとコピペ**
3. 末尾の `id="as-cta"` のリンク先（`href="#"`）を、申込フォームURL または LINE友だち追加URL に差し替え
4. 料金を変えたい場合は、同ファイル内の `FEE_TIERS` / `CLASS` / `OPT` の数字を編集

CSS・JSはすべて `#auc-sim` 配下にスコープしてあるため、テーマやLPtoolsの既存スタイルと干渉しません。

---

## 2. デザイン対応表（今のLP → LPtoolsブロック）

LPtoolsには使えるブロックが揃っています（`firstview`／`cta`／`voice`／`price`／`accordion`／`achievement`／`bullet`／`carousel` など）。今のデザインは次のように再現します。

| 今のLPのセクション | 使うLPtoolsブロック | 設定のポイント |
|---|---|---|
| ヒーロー（キャッチ＋実績数字＋CTA） | `firstview-05` ＋ `section`（背景画像） | 背景=車/オークション画像、バッジに「累計12,000台」等 |
| 選ばれる理由（3カード） | `bullet-01` または `carousel-content-01` | アイコン＋見出し＋本文の3枚 |
| ご利用の流れ（5ステップ） | `arrow` ＋ `section` の繰り返し | 番号・見出し・短文 |
| **料金シミュレーション** | `core/html`（カスタムHTML） | 本ガイド「1.」の埋め込み |
| 料金（価格帯テーブル） | `price` ＋ 表組み | 〜50万:39,800／…／300万〜:3% |
| お客様の声・実績 | `voice` ＋ `achievement-01` | 顔アイコン＋コメント、実績数字 |
| よくある質問 | `accordion-01` | Q&Aを開閉式で |
| CTA（申込促進） | `cta-01`〜`cta-04` | ボタン先を申込/LINEに |

### カラー・トーンの指定（デザイン統一用）
- メイン：ディープネイビー `#0e1b33` / アクセント：ゴールド `#c9a14a`（明）`#e0bd6a`
- 各ブロックの「文字色」「背景色」にこの値を入れると、シミュレーターとLP本体の見た目が揃います。

## 3. 全体の推奨構成（ハイブリッド）

```
WordPress（LPtools）
  └ 集客用LP（デザイン＋シミュレーター埋め込み）
        │  「無料登録」「オーダー」ボタン
        ▼
会員サイト（別途：会員制プラグイン or 既存の会員ページ）
        │  申込・かんたんオーダー
        ▼
GAS Web App → スプレッドシート登録 → LINE通知 → AI自動返信（既存の自動化）
```

- **LP（見せる部分）= LPtools**：更新が楽・テンプレ豊富。
- **会員・注文・自動化（動かす部分）= 会員ページ＋GAS**：すでに構築済みのものを接続。

---

## 4. このリポジトリの素材をLPtoolsへ移すには

- **画像/コピー/料金表**：本リポジトリの `site/index.html` がそのまま「原稿」になります。文言・数字をLPtoolsのブロックに転記してください。
- **シミュレーター**：`site/embed/simulator-embed.html`（カスタムHTMLに貼るだけ）。
- **会員サイト**：`site/mypage.html` / `login.html` と `gas/` の自動化はLPtoolsの外でそのまま使えます（LPのCTAから遷移）。

> ※ 本リポジトリの静的HTMLを、LPtools独自ブロック（`<!-- wp:ctwlpt-block/... -->`）の完全なテンプレートに自動変換することも技術的には可能ですが、ブロック仕様が独自で崩れやすいため、**「デザインは対応表でブロック再現＋シミュレーターはHTML埋め込み」**が最も確実でおすすめです。
