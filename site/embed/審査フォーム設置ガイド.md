# かんたんWEB審査フォーム 設置ガイド

新デザインの審査フォームを WordPress に設置する手順です。**「直接表示」と「ボタン→ポップアップ」の両方**に対応しています。

---

## 全体の考え方（かんたんに）

- **フォーム本体は `shinsa.html` の1ファイルだけ**です。
- 「直接表示」も「ポップアップ」も、この1ファイルを **iframe（枠）で呼び出すだけ**の仕組みにしてあります。
- なので、**フォームの中身を直したいときは `shinsa.html` だけ直せば、両方に同時に反映**されます。
- iframe で分離しているので、**WordPressのテーマやプラグインのデザインと絶対にケンカしません**。

```
　shinsa.html（フォーム本体）
　　├─ shinsa-embed.html        → ページに直接表示（審査ページ用）
　　└─ shinsa-popup-embed.html  → ボタンで開くポップアップ（LP・トップ用）
```

---

## STEP 0：まず `shinsa.html` をネット上に置く（1回だけ）

`shinsa.html` を、URLでアクセスできる場所にアップロードします。方法はどれか1つでOK：

- **おすすめ：** サーバーのフォルダに置く（例：`https://carmelonline.jp/form/shinsa.html`）
- WordPressのメディア／ファイルアップロード系プラグインで置く
- 別のホスティングに置く

置いたら、その **URLをメモ**しておいてください（次で使います）。

---

## パターン①：ページに直接表示（今の審査ページと同じ）

1. 審査フォームを表示したい**固定ページを編集**
2. 「＋」→ **「カスタムHTML」ブロック**を追加
3. `shinsa-embed.html` の中身を**丸ごと貼り付け**
4. 貼り付けた中の
   ```
   data-src="https://carmelonline.jp/form/shinsa.html"
   ```
   の URL を、**STEP 0 でメモした `shinsa.html` のURL**に書き換える
5. 公開

→ ページを開くとフォームがそのまま表示されます。ステップの切り替えや条件分岐で高さが変わっても、**枠の高さは自動で伸縮**します。

---

## パターン②：ボタンを押すとポップアップで表示（LP・トップページ向け）

1. ボタンを置きたいページ／LPを**編集**
2. 「＋」→ **「カスタムHTML」ブロック**を追加
3. `shinsa-popup-embed.html` の中身を**丸ごと貼り付け**
4. 貼り付けた中の
   ```
   data-src="https://carmelonline.jp/form/shinsa.html"
   ```
   の URL を、**STEP 0 でメモした `shinsa.html` のURL**に書き換える
5. （任意）ボタンの文言 `かんたんWEB審査を始める` を好きな言葉に変更
6. 公開

→ ページには**ボタンだけ**が表示され、押すとフォームが**画面の上に重なって開き**ます。背景クリック・×ボタン・Escキーで閉じられます。

---

## よくある質問

**Q. 両方のページに置いても大丈夫？**
はい。専用の審査ページには①、トップやLPには②、と併用できます。どちらも同じ `shinsa.html` を見ているので中身は常に同じです。

**Q. フォームの項目や文言を変えたい**
`shinsa.html` を編集して再アップロードすれば、①②の両方に反映されます。

**Q. 送信先はどうなっている？**
元のフォームと同じ仕組み（WordPress側の `carmel_shinsa_submit`）に送る作りを引き継いでいます。`shinsa.html` を **WordPress内のページ**として動かす場合はそのまま送信できます。静的ファイルとして別の場所に置く場合は、送信先だけ別途つなぎ込みが必要です（ご相談ください）。

**Q. スマホでも大丈夫？**
①はスマホでも高さ自動調整。②はスマホでは全画面のポップアップになります。どちらもスマホ最適化済みです。
