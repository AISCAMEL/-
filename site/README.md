# AUC-AGENT — 車のオークション代行 ランディングページ

中古車の**オークション代行（業者オークションの入札・落札代行）**サービス向けのランディングページ一式です。
サイト上で**乗り出し総額のシミュレーション**ができ、**会員サイト（マイページ）**から**かんたんにオーダー**できるイメージを再現しています。

## サイトマップ / 構成

| ファイル | 役割 |
|---|---|
| `index.html` | **購入代行**LP（ヒーロー／理由／流れ／総額シミュレーション／料金／安心・保証／実績・声／FAQ／問合せ／会社情報） |
| `sell.html` | **出品代行（売却）**LP＋**出品手取りシミュレーション** |
| `shipping.html` | **陸送費シミュレーター**（出発↔納車エリア×車種） |
| `manual.html` | **出品の仕方マニュアル**（会員限定） |
| `partners.html` | **パートナー募集**（加盟店/紹介・別サイト想定）＋申込フォーム |
| `login.html` | 会員登録・ログイン（タブ切替） |
| `mypage.html` | 会員サイト。**購入/出品 両対応**の状況確認＋かんたんオーダー＋出品申込（即時見積り） |
| `assets/css/style.css` | デザインシステム（ネイビー×ゴールド、レスポンシブ対応） |
| `assets/js/simulator.js` | 計算ロジック（`window.AucSim`）：購入総額 `calculate`／陸送 `estimateShipping`／出品手取り `estimateSell` |
| `assets/js/app.js` | 会員サイトのロジック＋**バックエンド送信**（`ENDPOINT`にGAS URLを設定するとシート登録/LINE/AIへ連携） |

### 料金モデル（市場調査ベース）
- 購入代行：落札価格帯別の定額（〜50万 ¥39,800 …／300万〜 3%）
- 出品代行：落札価格の **2.5%** と 定額 **¥22,000** の高い方 ＋ 出品料・成約料 ¥11,000
- 陸送：地域ゾーン距離×車種（北海道・沖縄は航送加算）
- ※ 数値は `assets/js/simulator.js` の定数（`FEE_TIERS`/`SELL_RATE`/`AREAS` 等）に集約

LPには **選ばれる理由／利用の流れ／料金シミュレーション／料金／安心・保証／落札実績・お客様の声／FAQ／お問い合わせ・LINE／会社情報・特商法** を実装済み（実データに差し替え可のプレースホルダーは黄色注記）。

## バックエンド自動化（`../gas/`）

申込を **GAS Web App → スプレッドシート登録 → LINE通知 → AI要約** まで自動化します。
セットアップは [`../gas/セットアップ手順.md`](../gas/セットアップ手順.md) を参照。

| ファイル | 役割 |
|---|---|
| `gas/Config.gs` | 設定（シートID・LINEトークン・OpenRouterキー） |
| `gas/Setup.gs` | `setupAll()` で3シートを自動作成 |
| `gas/WebApp.gs` | `doPost()`：LPからの登録/オーダー/問い合わせを受信 |
| `gas/Line.gs` | スタッフへLINE通知 |
| `gas/AI.gs` | OpenRouterで申込を要約・優先度づけ（キー無しは簡易ルール） |

## 主な機能

- **総額シミュレーション** — 落札予算・車両クラス・納車エリア・オプションを選ぶと、代行手数料／陸送費／諸費用込みの「乗り出し総額」と、店頭購入との差額（おトク額）の目安をその場で表示。
- **会員サイト** — 登録／ログイン後、マイページで注文状況（進捗バー・ステータス）を確認。
- **かんたんオーダー** — 希望車種・予算を入力するだけ。シミュレーターと同じ計算ロジックで即時見積もりを出し、ワンクリックでオーダーを登録。

## 料金モデル（全国平均ベース）

代行手数料は **落札価格帯別の定額制**。国内の代行業者の公開料金の中央値に合わせています。

| 落札価格帯 | 代行手数料（税込） |
|---|---|
| 〜50万円 | ¥39,800 |
| 50〜100万円 | ¥49,800 |
| 100〜150万円 | ¥59,800 |
| 150〜200万円 | ¥69,800 |
| 200〜300万円 | ¥79,800 |
| 300万円〜 | 落札価格の 3% |

- 割増：SUV・大型 +¥5,000／輸入車 +¥20,000
- 別途実費：オークション落札料 ¥11,000、リサイクル料等 ¥12,000〜、陸送費 ¥20,000〜（エリア別）
- 任意オプション：名義変更・登録 ¥25,000／車庫証明 ¥12,000／12ヶ月点検 ¥35,000／6ヶ月保証 ¥28,000

料金は `assets/js/simulator.js` の `FEE_TIERS` / `CLASS` / `OPTIONS` に集約しているため、ここを編集すればLP・シミュレーター・会員サイトすべてに反映されます。

## ローカルで確認する

静的サイトなので、サーバー不要でそのまま開けます。

```bash
# 例: 簡易サーバーで起動（任意）
cd site
python3 -m http.server 8000
# → http://localhost:8000 をブラウザで開く
```

`index.html` をブラウザで直接開いてもOKです。

## 注意（デモ実装について）

- 会員登録・ログイン・注文データは、デモのため**ブラウザのlocalStorage**に保存されます（実際の認証・サーバー保存は行いません）。
- 料金・手数料・おトク額はすべて**参考値**です。実運用ではバックエンドAPI／決済／オークションシステム連携への置き換えを想定しています。
