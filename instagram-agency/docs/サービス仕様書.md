# サービス仕様書

## 目的
運用オペレーションのブレを最小化し、担当者・外注チェッカーが誰であっても
同じ品質で納品できる状態を作る。

---

## 1. オンボーディングフロー（契約後 5営業日以内）

| Day | 作業 | 実施者 | 納品物 |
|-----|-----|--------|-------|
| Day 0 | 契約書締結・初期費用決済 | 自分 | 契約書PDF |
| Day 1 | キックオフMTG（60分・Zoom）| 自分 | ヒアリングシート |
| Day 1〜2 | 業種プロンプトのカスタマイズ | 自分 | Notion上のプロンプト集 |
| Day 2〜3 | 現状アカウント分析 | 自分 | 分析レポート |
| Day 3〜4 | 3ヶ月分の投稿カレンダー案作成 | AI＋自分 | Googleスプレッドシート |
| Day 4 | ハッシュタグ資産 100件構築 | AI＋自分 | Notion DB |
| Day 5 | 顧客レビュー・修正・確定 | 自分 | 承認済カレンダー |
| Day 5 | 初回投稿予約セット | 自分 | Meta Business Suite画面キャプチャ |

**ヒアリングシートの項目**
- ブランドの一言説明
- ターゲット像（年代、性別、地域、悩み）
- NGワード・NG画像
- 過去に反応が良かった投稿URL 3件
- 競合と参考にしているアカウント 各3件
- 月次の販促イベント（セール、新商品、季節）
- 使用可能な素材の共有方法（Googleドライブ想定）

## 3. AI投稿案の生成仕様

### プロンプトの構造
```
【役割】あなたは{業種}のInstagram運用担当者です。
【ブランド】{ヒアリングから取得したブランド概要}
【ターゲット】{ペルソナ}
【NG】{NGワード・トピック}
【投稿目的】{今回の目的：認知/教育/販促/共感}
【投稿タイプ】{フィード / リール / カルーセル}
【素材】{添付画像や商品情報}

以下のフォーマットで出力してください：
1. キャプション本文（300字以内）
2. ハッシュタグ 20個（3〜5万投稿ボリューム帯 中心、大中小のミックス）
3. 冒頭1行目のフック案 3種
4. 使用推奨時間帯
5. 想定リアクションと成功指標
```

### 品質チェック観点（人チェック工程）
1. NGワード混入していないか
2. ブランドトーンから逸脱していないか
3. 誇大表現・薬機法・景表法違反リスク
4. ハッシュタグの過剰重複
5. 前月・前々月と同じ切り口の使い回しになっていないか

## 4. リール台本の提供仕様（Standard 月2本、Pro 月4本）

**構成**
1. フック（0〜1秒、視聴継続の意思決定ポイント）
2. 前振り（1〜3秒、問題提起）
3. 本編（3〜20秒、具体例3つ想定）
4. 締め（20〜30秒、CTA）

**納品物**
- Googleドキュメントに台本と撮影メモ
- Pro プランは絵コンテ簡易版（各カット1文説明＋参考リンク）

## 5. 月次レポートの仕様

### Light版（1ページ）
- フォロワー増減
- リーチ・インプレッション合計
- 投稿別いいね・保存
- 一言コメント

### Standard版（3ページ）
- 上記
- 週次推移グラフ
- 投稿別スコア（保存率、リーチ、リアクション率）
- 改善提案 3件

### Pro版（5ページ以上）
- 上記
- 競合3社比較（投稿頻度、フォロワー成長率、エンゲージメント率）
- 時間帯別リーチのヒートマップ
- 次月企画テーブル 10案
- 定量目標と現状のギャップ分析

## 6. SLA（サービスレベル）

| 項目 | 基準 |
|------|------|
| 投稿案の共有 | 予定投稿日の3日前まで |
| 顧客からの承認〜投稿予約 | 承認後24時間以内 |
| チャット返信 | 平日9〜18時、原則2営業時間以内 |
| MTG 日程調整 | 依頼から2営業日以内に提示 |
| 月次レポート | 翌月5営業日以内 |
| BAN/凍結対応 | 対象外（規約明記）|

**SLA未達時**：翌月分の10%相当の返金 or サービス延長。

## 8. 秘密保持と情報管理

- 顧客ごとに専用の共有フォルダを分離
- 素材保存先を顧客専有し、他顧客と絶対に混在させない
- Instagramログイン情報は原則預からず、**Meta Business Suiteのオーナー招待方式**を推奨
- 万一預かる場合はパスワードマネージャで暗号化保管、契約終了時に完全削除

---

## 9. 撤退基準

以下いずれかに該当する顧客は「サービス品質のため契約更新見送り」を検討：
1. 支払い遅延が2回発生
2. ハラスメントに近いコミュニケーション
3. NGワード確認や規約違反ラインを繰り返し要求（誇大表現、他社誹謗中傷など）
4. 5営業日以上、投稿案への承認応答が繰り返しない

契約書に「相互解除条項」を入れておく。
