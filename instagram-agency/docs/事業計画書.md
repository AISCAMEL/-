# Instagram運用代行「格安AI運用会社」 事業計画書

## 1. 事業概要

### 1.1 サービス名（仮）
AIグラム / Insta AI Partner

### 1.2 事業内容
中小企業・個人事業主・店舗向けに、**AIによる投稿案の自動生成**を軸とした
Instagram運用代行サービスを提供する。人力工数を最小化することで、
業界水準（月額10〜30万円）に対し **月額1〜4万円台** の圧倒的低価格を実現する。

### 1.3 ミッション
「投稿を続けたいが手が回らない」全ての小規模事業者に、月額1万円台から
プロ品質のInstagram運用を届ける。

### 1.4 事業形態
- 開始形態: 個人事業（合同会社化は月商50万円到達後を目安）
- 拠点: 完全リモート（バーチャルオフィス契約でOK）
- 開業初期投資目安: 20万円以下（後述）

---

## 2. 市場と課題

### 2.1 市場背景
- 日本国内の月間アクティブアカウント数は依然3,000万超規模。SMBのMarketing予算内訳でも
  SNS運用が定常経費化している。
- 一方、既存の運用代行相場は **月額15〜30万円** が中心。中小・個人店舗は「必要だが高すぎる」
  というギャップが構造的に存在する。
- ChatGPT / Claudeの浸透により、キャプション・ハッシュタグ・企画立案などの
  従来「人の作業」だった領域を、AIで80〜90%まで自動化できるようになった。

### 2.2 ターゲット顧客
| セグメント | 特徴 | 想定月額 |
|-----------|------|---------|
| 個人店舗（美容室・サロン・カフェ・整体等）| 予算月2〜3万円、投稿を続けられない | ¥9,800 〜 ¥19,800 |
| 個人事業主・フリーランス（士業、コーチ）| 集客したいが本業で手が回らない | ¥9,800 〜 ¥19,800 |
| EC / D2C 小規模ブランド | 商品数少、投稿の質と継続性を欲する | ¥19,800 〜 ¥39,800 |
| 中小企業のマーケ担当兼務者 | 「片手間運用」を外に出したい | ¥19,800 〜 ¥39,800 |

### 2.3 顧客の主な悩み（ヒアリング仮説）
1. 「毎日投稿するネタが尽きる」
2. 「他社に頼むと月20万円と言われて即諦めた」
3. 「自分でやるとハッシュタグ選定・キャプションで1投稿1時間かかる」
4. 「効果が出ているのか分からない」

上記いずれもAI＋軽い人チェックで解決可能。

## 3. サービス構成（詳細は「サービス仕様書.md」）

### 3.1 提供の中核
1. AIによる投稿案（キャプション＋ハッシュタグ＋画像プロンプト）を月20〜60本自動生成
2. 人による最終チェック（月4〜12本）
3. 月次分析レポート（リーチ・フォロワー増減・投稿別成果）
4. 予約投稿・スケジューリング支援

### 3.2 提供しない（意図的スコープ外）
- 撮影・モデル手配（別途オプション or 外注紹介）
- 有料広告の運用代行（別サービスとして後日展開）
- コメント・DMのリアルタイム24h対応（Standard以上でBiz時間帯のみ対応）

### 4.1 価格帯（詳細は「料金プラン.md」）

- Pro: ¥39,800/月（Standard＋競合分析、DMテンプレ、月1MTG）

## 6. 差別化（なぜ格安が可能か）

競合が月額20万円取っている工数のうち、**AIで自動化できる領域が7割**を占める。
| 業務 | 従来（1顧客あたり月工数）| AI化後 |
|------|-----------------------|-------|
| キャプション作成（20本）| 10時間 | 1時間（レビューのみ）|
| ハッシュタグ選定 | 5時間 | 0.5時間 |
| 投稿ネタ企画 | 5時間 | 1時間 |
| 分析レポート | 3時間 | 0.5時間 |
| **合計** | **23時間** | **3時間** |

23時間 → 3時間の圧縮により、時間単価¥3,000で換算すると
1顧客あたり原価が¥69,000 → ¥9,000へ低下。これが価格破壊の根拠。

---

## 7. リスクと対応

| リスク | 対応 |
|-------|-----|
| AI生成品質のブレ | 業種別プロンプトテンプレを整備、人チェック工程を必ず挟む |
| 顧客のInstagram BAN | 「BAN補償なし」を規約明記。運用ガイドラインをオンボード時に説明 |
| 低単価による疲弊 | Light枠は各月上限20社で締切。以降はStandard以上のみ受付 |
| 顧客解約率 | 3ヶ月縛りなし＋成果レポートで継続動機を作る。初月成果保証は謳わない |
| 競合の値下げ追随 | AIプロンプト資産＋業種別ノウハウ蓄積で差別化。安さだけを競わない |

## 8. 立ち上げロードマップ（3ヶ月）

### 月1（準備）
- サービスサイト（LP）公開
- 料金・契約書・利用規約を確定
- AIプロンプトテンプレ 5業種分作成（美容・飲食・EC・士業・サロン）
- 自社Instagram / X 開設と発信開始
- 決済導線（Stripe / 銀行振込）整備

### 月2（初期受注）
- クラウドソーシングで最初の3〜5社受注（テスト価格 ¥5,000〜）
- 運用オペレーションを回し、工数を計測
- 実績スクショ・数値を蓄積

### 月3（拡大）
- 通常価格で LP からの受注に移行
- 実績記事・お客様の声を公開
- 月商20〜30万円のライン到達を目標
