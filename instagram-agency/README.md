## 事業のコンセプト（要約）

- **サービス名（仮）**: AIグラム / Insta AI Partner
- **顧客**: 中小企業、個人店舗、フリーランス、小規模EC
- **提供**: AI投稿案の自動生成 + 人チェック + 月次分析
- **価格**: 月額 ¥9,800 / ¥19,800 / ¥39,800（3プラン）
- **初期費用**: ¥29,800（1回のみ）
- **契約縛り**: なし（月次更新）
- **粗利率**: Standardで36〜55%（オペ最適化後）
- **損益分岐**: 契約4社前後

## なぜ格安が可能か（一言）

キャプション・ハッシュタグ・分析レポートといった「AIが得意な領域」の作業を
自動化することで、1顧客あたりの月工数を **23時間 → 3時間** に圧縮。
浮いた工数を価格に還元し、業界相場（月20万円前後）に対し
**約1/10** の価格を実現する。

## 次のステップ（推奨順）

1. サービス名の商標調査 → 屋号決定
2. LP制作（既存の `../site` を参考にHTML1枚から）
3. 決済導線の準備（Stripe）
4. 自社Instagram / X の開設と発信開始
5. プロンプトテンプレを5業種分（美容、飲食、EC、士業、サロン）作成
6. クラウドソーシングでのテスト販売開始
