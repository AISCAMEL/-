## 2. 料金・手数料（[要確認] 市場調査で確定）

現状LPは購入代行の手数料を「落札価格帯別の定額」で設定済み。出品代行・ローン・オプションは**市場調査で確定**する。

| 区分 | 暫定 | 確定方法 |
|---|---|---|
| 購入代行 手数料 | 〜50万:3.98万／…／300万〜:3%（実装済） | [要確認] 市場調査で最終化 |
| 出品代行 手数料 | **落札価格帯別の定額**（〜50万¥35,000／50-100万¥45,000／以降+¥20,000前後／900〜1000万¥220,000／1000万超は応相談）＋出品料¥11,000 | ✅ オーナー確定 |
| 出品 不成約時 | 流札・再出品料 **無料** | ✅ 確定 |
| ローン金利・手数料 | オリコ規定（2.9〜8.9%目安） | オリコ連携で確定 |
| オプション | 下記「オプション一覧」 | 市場調査 |

> アクション：競合の購入/出品の手数料表を継続調査し、本表を確定 → `simulator.js` の定数に反映。

## 4. オプションの充実（[要確認] 価格）

購入・出品それぞれで選べるオプション。シミュレーターと会員オーダーに反映。

- 共通：名義変更・登録代行／車庫証明取得／陸送（エリア別）／12ヶ月点検・整備／保証パック
- 購入：希望ナンバー／コーティング／ドラレコ等取付／下取り同時査定
- 出品：出品前 簡易点検・撮影代行／室内クリーニング／デジタル出品票 制作（後述）／代理搬入
- **[要確認]** 各オプションの価格（市場調査）。

## 5. 業務フロー

### 5-A. 購入代行（落札）フロー
1. 会員登録 → 2. 条件オーダー → 3. 車両提案・**ローン審査（希望者・オリコ）** → 4. 入札代行 → 5. 落札 → 6. **契約書**締結・**入金** → 7. 陸送・名義変更 → 8. 納車 → 9. **クレーム期間**対応 → 10. アフター

### 5-B. 出品代行（売却）フロー ★新規
1. 会員登録 → 2. 出品申込（車両情報入力） → 3. **落札・出品シミュレーション**で想定額提示 → 4. **契約書**締結 →
5. **デジタル出品票を顧客情報から制作 → 郵送** → 6. **ユーザーが車両を会場へ搬入（会場搬入）** →
7. オークション出品・成約 → 8. 成約代金の**入金**（手数料控除） → 9. **クレーム期間**（落札者対応・**違約金**条件） → 10. 完了

> 出品票：顧客が入力した車両情報（車検証・走行・装備・状態申告）をもとに**デジタル出品票（PDF）を自動生成**し郵送。ユーザーは指定会場に自走/積載で搬入。

### 主要論点（フロー共通）
- **会場搬入**：対応会場リスト・搬入期限・必要書類 **[要確認]**
- **クレーム期間**：例 落札後○日以内（出品車の状態相違クレーム受付期間） **[要確認] 日数**
- **入金**：購入＝事前/分割、出品＝成約後○営業日入金 **[要確認] サイト・方法**

## 7. 会員サイトの充実化

現状（注文状況＋かんたんオーダー）に加えて：

- マイページ拡張：購入/出品の**両タブ**、進捗ステータス（出品票郵送→搬入→出品→成約→入金…）

- 決済/入金：入金状況の可視化

## 9. パートナー制度（別サイト）

- 別ドメイン/別LPで**加盟店・紹介パートナーを誘致**。
- 内容：報酬体系・募集要項・申込フォーム → 本部DBへ連携。
- 既存仕様の「フランチャイズ（本部＋加盟店）」構造を踏襲。
- **[要確認]** 報酬モデル（紹介料 定額/成果報酬）、対象（個人/法人）。
