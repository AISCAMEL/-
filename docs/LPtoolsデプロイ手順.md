# LPtools で AUC-AGENT のLPを公開する（デプロイ手順書）

運営：合同会社アイズ（AUC-AGENT）／最終更新：2026-06-16

> **方式**：WordPress ＋ LPtools（Catch the Web 社の有料プラグイン）でLPを構築・公開します。
> 会員サイト・自動オーダー・各種申込は LPtools の守備範囲外のため、**既存の会員ページ（`site/mypage.html` `login.html`）＋GAS自動化**へCTAで接続する**ハイブリッド構成**です。
>
> ⚠️ 本手順は貴社のWordPress管理画面で操作いただく前提です（当方からは貴社WPへ直接アクセスできません）。

---

## 0. 準備するもの
- WordPress サイト（管理者ログイン）
- **LPtools プラグイン**のライセンス（Catch the Web）
- 本リポジトリの素材（`site/embed/` 配下）
  - `LP_lptools_import.html` … インポート用ブロックマークアップ（見出し・料金表・FAQ・CTA・会社情報）
  - `simulator-embed.html` … 購入総額シミュレーター（カスタムHTMLに貼るだけ）
  - `sell-simulator-embed.html` … 出品手取りシミュレーター（出品代行LP用）
  - `LP原稿_LPtools用.md` … ヒーロー・流れ・お客様の声の転記原稿
- 差し込むURL2つ：**申込ページURL**（例 `login.html` の公開先）と **LINE友だち追加URL**

---

## STEP 1. LPtools を有効化
1. WordPress管理画面 →「プラグイン」→ LPtools をインストール・有効化（ライセンス認証）。
2. 「LPtools」メニューが出ることを確認。

## STEP 2. LP用の固定ページを作る
1. 「固定ページ」→「新規追加」。タイトル例：`購入代行LP`。
2. 右側「ページ属性」→ テンプレートを **LPtools のLP用テンプレート（フルワイド／ヘッダーフッター無し系）** に設定。
   - これでテーマのヘッダー/フッターに邪魔されず、LP単体で表示されます。

## STEP 3. ベースを一括インポート（コードエディター）
1. 編集画面右上「⋮（オプション）」→ **「コードエディター」** に切り替え。
2. `site/embed/LP_lptools_import.html` の中身を**丸ごと貼り付け**。
3. **「ビジュアルエディター」に戻す** → 見出し／料金表／FAQ／CTA／**会社情報・特商法**がブロックとして展開されます。
   - 「ブロックのリカバリー」表示が出たら各ブロックで「リカバリーを試行」を押せば直ります。

## STEP 4. 料金シミュレーターを埋め込む
1. 「サイト上で『乗り出し総額』をその場で計算」見出しの下にある **カスタムHTML（wp:html）ブロック**（点線の枠）を選択。
2. 枠内のプレースホルダーを消し、`site/embed/simulator-embed.html` の中身を**丸ごと貼り付け**。
3. CSS/JSは `#auc-sim` 配下にスコープ済みなのでテーマと干渉しません。
   - 料金を変えたい場合は同ファイル末尾の `FEE_TIERS` / `CLASS` / `OPT` を編集。

## STEP 5. ヒーロー・流れ・お客様の声を追加（ビジュアル）
`LP原稿_LPtools用.md` の②③④⑦を見ながら、画像が絡むブロックをビジュアルで追加：
- ② ヒーロー：`firstview-05` ＋ 背景section（キャッチ・実績数字・**24時間受付バッジ**・CTA）
- ③ 選ばれる理由：`bullet-01-b` ×3
- ④ ご利用の流れ：`numbering-content-01` ×5
- ⑦ お客様の声：`voice-01-b` ×3 ＋ `achievement-01`
  - ⚠️ お客様の声・実績数字は**サンプル**です。**実データ（許可を得た声・実数）に差し替え**てください（景表法／サクラ表記の回避）。

## STEP 6. CTA・リンク先を設定
- 各CTAボタン（ヒーロー・料金下・最下部）の遷移先を設定：
  - 「無料会員登録／オーダー」→ **申込ページURL**（`login.html` の公開先）
  - シミュレーター内CTA（`id="as-cta"`、`href="#"`）→ 申込 or LINE URL
- LINE友だち追加ボタン → **LINE公式アカウントURL**（例 `https://lin.ee/xxxx`）

## STEP 7. カラー・最終調整
- メイン `#0e1b33`（ネイビー）／アクセント `#c9a14a`・`#e0bd6a`（ゴールド）を各ブロックの文字色・背景色に。
- スマホ表示を確認（シミュレーターは760px以下で1カラムに自動切替）。

## STEP 8. SEO・公開
1. SEOプラグイン（All in One SEO / Yoast等）で：
   - タイトル：`車のオークション代行｜相場よりおトクに — AUC-AGENT`
   - ディスクリプション・**OGP画像**（`site/assets/img/ogp.png` をメディアにアップして指定）
2. パーマリンク設定（例 `/lp/purchase/`）。
3. **公開**。

## STEP 9. 出品代行LP（任意・同手順）
同じ流れで2枚目の固定ページ（例 `出品代行LP`）を作成。原稿は `site/sell.html` の文言を流用。
シミュレーターは**出品手取り版**を使います：STEP4と同じ手順で、カスタムHTMLに
`site/embed/sell-simulator-embed.html` の中身を貼り付け（購入用 `#auc-sim` と別IDなので**同一サイトに同時設置OK**）。

## つまずき対策（LPtoolsでよくある詰まり）
| 症状 | 対処 |
|---|---|
| インポート後「このブロックにはエラーが含まれています」 | 各ブロックの「⋮」→**「ブロックのリカバリーを試行」**。プラグインのバージョン差が原因で、見た目は復旧します。 |
| 貼り付けたHTMLが**コードのまま**表示される | コードエディターに貼った後、必ず**ビジュアルエディターに戻す**。シミュレーターは「カスタムHTML（wp:html）」ブロックに貼る（段落ブロックに貼らない）。 |
| シミュレーターが**動かない/二重に出る** | 1ページに同じ `#auc-sim` を2回貼らない。購入は `#auc-sim`、出品は `#auc-sell-sim`。キャッシュ系プラグインがあればキャッシュ削除。 |
| LPに**テーマのヘッダー/フッターが出てしまう** | ページ属性のテンプレートを**LPtoolsのLP用（フルワイド／ヘッダーフッター無し）**に変更。 |
| 文字や背景の**色が変わらない** | ブロックを選択→右サイドバー「色」で文字/背景を `#0e1b33`・`#c9a14a` に。テーマCSSが優先する場合はLPtoolsの「カスタムCSS」で上書き。 |
| スマホで**レイアウトが崩れる** | プレビューのモバイル表示で確認。シミュレーターは760px以下で自動1カラム。LPtoolsブロックは各ブロックの「レスポンシブ」設定を確認。 |
| ボタンを押しても**何も起きない** | 各CTAの遷移先URL（申込/LINE）が空のまま。STEP6で設定。シミュレーター内CTAは `id="as-cta"`/`id="ss-cta"` の `href="#"` を実URLに。 |
| 更新したのに**反映されない** | キャッシュプラグイン/サーバーキャッシュ/ブラウザのスーパーリロード（Ctrl+F5）。 |

## 会員サイト・自動化との接続（ハイブリッド）
```
LPtools（集客LP）──CTA──▶ 会員ページ（login.html / mypage.html）──申込──▶ GAS Web App
                                                          ├─ スプレッドシート登録
                                                          ├─ LINE / Slack 通知
                                                          ├─ 出品票PDF・オリコ申込書PDF（メール）
                                                          └─ 相場見積りの回答反映
```
- 会員ページ＋GASのデプロイは `docs/デプロイ手順.md` の B章（実行ガイド）参照。
- LPと会員ページのドメイン/配置は、同一サイト内のパスでも、別ホスティング（Netlify等）でもOK。CTAのURLで繋ぎます。
