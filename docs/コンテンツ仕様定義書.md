# 合同会社アイズ コーポレートサイト コンテンツ仕様定義書（WordPress版）

> 目的：実際のコピー文を書き始める前に、**「各ページに何を・どんな構造で・どの項目を載せるか」** を確定させる。
> 本書が「中身の設計図」となり、以降はこの枠を文章で埋めていく。
> `（要記入）`＝お客様にご提供・ご確認いただく項目。`（こちらで作成）`＝コピーを当方で起案。

## 0. 前提・全体方針

| 項目 | 定義 |
| --- | --- |
| プラットフォーム | WordPress（現行サイトを継続利用） |
| 目的 | 「何の会社か／誰向けか／何を頼めるか／なぜアイズか」を即伝え、**問い合わせにつなげる** |
| サービス見せ方 | **4事業** に集約。**自動車事業を主力（訴求の約8割）**、アプリ事業を約2割、加えてGPS事業・FC事業を独立の柱として提示 |

| トーン | 未来志向・誠実・専門的・わかりやすい（BtoB） |
| 主要CV | お問い合わせ／無料相談 |

## 5. 確定事項・残論点

### 5-1. 確定済み（2026-06-09）
1. **事業構成**：4事業に再編。**自動車事業を主力（訴求 約8割）**、アプリ事業 約2割、GPS事業・FC事業を独立柱として追加。
2. **自動車事業の構成**：「**販売・買取・リース**」の3本立てで確定（メンテ等は含めない）。
3. **アプリ事業**：これまでのWeb制作・マーケティング・システム開発を**アプリ事業に統合**。自社アプリ「App R」「EX」を展開（内容は要確認）。

5. **主軸の強調度**：自動車を主力としてトップ・一覧で大きく強調。
6. **実績**：サンプル運用で開始（実績確定後に差し替え）。

### 5-2. 残論点（情報提供待ち）
1. アプリ事業「App R」「EX」の正式名称・内容。
2. GPS事業の具体的なサービス内容（提供内容・対象・特長）。
