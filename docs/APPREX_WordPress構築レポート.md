# 📦 APPREX WordPress 構築レポート

**日付**: 2026-06-09
**対象**: 現行 APPREX 静的サイト（Cloudflare Pages 版・合同会社アイズ）→ WordPress 新規構築

---

## 1. 実施内容

現行サイトの実データ（HTML/CSS/JS/画像/コピー）を精査し、WordPress 向けにブラッシュアップして自作テーマ `apprex` を再構築。**ローカルに WordPress を新規構築**してテーマを実装・検証し、アップロード用パッケージ（`dist/apprex-theme.zip`）を作成した。

## 2. 検証環境（このサンドボックスで実構築）

| 項目 | 内容 |
|------|------|
| WordPress | 6.7.1（GitHub ミラーから取得 ※ wordpress.org はネットワーク許可外） |
| PHP | 8.4.19（ビルトインサーバー + wp-cli ルーター） |
| DB | SQLite（`sqlite-database-integration` ドロップイン） |
| URL | http://127.0.0.1:8088 |

> ネットワークはアローリスト制で wordpress.org が遮断されているため、コア・wp-cli・SQLite プラグインはすべて GitHub から取得。本番環境では通常どおり wp-admin からテーマ zip をアップロードして導入できる。

## 3. 検証結果

- 全ページ HTTP **200**：`/` `/features/` `/functions/` `/pricing/` `/faq/` `/cases/` `/hp-creation/` `/company/` `/contact/` `/free-trial/`
- フロントページに実コピー・キャンペーンバー・統計・料金・Instagram・チャットボット・導入事例カード5件を表示
- 導入事例アーカイブにカード5件、シングルにメタ情報（業種/成果/期間）表示
- `wp-content/debug.log` に **PHP fatal/error なし**
- **self-seeder をクリーン DB で検証**：テーマ有効化のみで固定ページ9・事例5（画像サムネ付き）・静的フロントページ・グローバルメニュー（primary）を自動生成

※ ビジュアルのスクリーンショットは、ヘッドレスブラウザ（Chromium）のバイナリ取得がアローリスト外で取得不可のため未取得。HTTP ステータス＋レンダリング HTML のマーカー検証で代替確認済み。

## 4. ブラッシュアップ（現行サイトからの修正）

| 項目 | 現行 | 対応 |
|------|------|------|
| 料金の不整合 | `pricing.html` が ¥30,000/¥80,000、電話 03-XXXX-XXXX、©2024、誤記「アプリックス」等の stale テンプレート | README/トップの正データ（アプリ開発 Trial¥19,800・Start¥39,800・Business¥59,800、初期費用0円、最低12ヶ月）に統一。制作代行 ¥100,000〜、HP制作 月額9,800円〜 も反映 |
| 配色・書体 | 旧紫系の名残 | 要件「案A 爽やかなブルー系」を正式採用（#3B82F6 ほか）、ヒラギノ角ゴ系 |
| 電話番号 | 廃止方針 | 全ページ非表示を踏襲（CSS でも `tel:` を非表示） |
| 「即日公開」 | 誤解を招く表現 | 「スピード公開（最短2週間）」に統一、即日表現は不使用 |
| チャットボット | Zapier | 全ページ右下に常設（クリックで遅延読み込み） |
| ページ重複 | index.html と about/blog-1〜6/services/terms が同一バイト（パッケージ不具合） | WordPress では各ページを正規の固定ページ／CPT として分離 |

## 5. 成果物

- `wp-content/themes/apprex/` … テーマ本体（self-seeder 内蔵）
- `dist/apprex-theme.zip` … 管理画面からアップロード可能なインストールパッケージ
- 本レポート＋テーマ `README.md`

## 6. 本番反映の手順（お客様作業）

1. 本番 WordPress に `apprex-theme.zip` をアップロードして有効化（自動でページ・事例・メニュー生成）
2. ACF / Contact Form 7 等を導入し、フォームのショートコードを `free-trial` / `contact` ページへ
3. パーマリンクを一度保存
4. 必要に応じてコピー微調整・画像の WebP 化
