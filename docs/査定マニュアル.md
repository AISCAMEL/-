# 車の査定マニュアル（AUC-AGENT／合同会社アイズ）

会員向け「車の査定マニュアル」の原稿。Webページ版は `site/appraisal.html`。

このマニュアルは、オートオークション（AA）での評価のしくみと、査定額を左右するポイントをまとめたものです。お客様が「自分の車がいくらで売れそうか」を理解し、正確な状態申告で減額・返品リスクを避けられることを目的としています。

## 1. 査定の5ステップ

1. **情報入力** … 年式・走行・装備・状態をマイページの出品申込フォームから入力
2. **相場照合** … 直近のオートオークション（AA）成約データと照合
3. **評価点想定** … 外装・内装のコンディションを仮判定
4. **想定額提示** … 落札想定レンジを提示
5. **手取り試算** … 手数料・オプション控除後の手取りを確認

> 査定額は「確定額」ではなく、直近のAA成約実績にもとづく**想定レンジ**です。最終額は当日の入札で決まります。

## 2. 査定に必要な情報

- メーカー・車名・グレード（車検証どおり）
- 初度登録年月（年式）
- 走行距離（直近メーター値）
- 修復歴の有無（正確に申告。不明なら「不明」）
- 車検残（満了日）
- 主要装備（ナビ・サンルーフ・レザー・4WD 等）
- 外装キズ・へこみ・内装の状態
- 純正／社外パーツ・改造の有無

> 車検証・メーター・車体全体の写真があると査定精度が大きく上がります。

## 3. オートオークション 外装評価点の見方

| 評価点 | 状態の目安 |
|---|---|
| S / 6点 | 新車・未使用車クラス。ほぼ無傷。 |
| 5点 | 年式が新しく内外装とも非常に良好。目立つキズなし。 |
| 4.5点 | 走行少なめで状態良好。小キズが数か所程度。 |
| 4点 | 使用感はあるが良好。市場で最も多い一般的なコンディション。 |
| 3.5点 | キズ・へこみがやや目立つ。内装にも使用感。 |
| 3点 | キズ・へこみ・内装劣化が複数。過走行車も多い。 |
| 2点 | 外装・内装ともに難あり。要補修レベル。 |
| 1点 | 不動・全塗装・改造多数など。特殊扱い。 |
| R / RA | 修復歴あり（事故歴・骨格部の修正・交換）。減額対象。 |
| 0点 / *** | メーター交換・改ざん、水没等で通常評価不可。 |

> **4点**が「ふつうに使われてきた良好な中古車」の基準点。4点を境に上下で査定額が段階的に変わります（会場により多少差あり）。

## 4. 内装評価の見方（A〜D）

| 内装評価 | 状態の目安 |
|---|---|
| A | 非常にきれい。喫煙なし・汚れなし。 |
| B | 良好。目立つ汚れ・へたりなし。 |
| C | 使用感あり。汚れ・スレ・小さな焦げ跡など。 |
| D | 汚れ・破れ・強い臭い等が目立つ。要クリーニング。 |

> 出品前の**内外装クリーニング（サイズ別 ¥12,000〜¥30,000）**で内装評価が上がり、落札額の底上げにつながる場合があります。喫煙・ペットの臭いは減額要因になりやすいので、消臭も有効です。

## 5. 車両検査票のキズ表示記号（展開図）

車両検査票には、車を上から見た**展開図（車両図）**に、キズ・へこみの位置と種類が記号で記入されます。落札者はこの記号でコンディションを把握します。

| 記号 | 意味 |
|---|---|
| A | キズ（A1＝小／A2＝中／A3＝大／A4＝特大） |
| U | へこみ（U1＝小／U2＝中／U3＝大） |
| W | 補修跡・板金塗装跡（W1〜W3）／波打ち |
| S | サビ（S1＝小／S2＝中／S3＝大） |
| C | 腐食（サビによる穴・進行した劣化） |
| P | 塗装の劣化・色あせ・塗装済み |
| B | バフ傷（磨きによる細かなキズ） |
| X | 要交換（損傷が大きく交換が望ましい部位） |
| XX | 交換済みパネル |
| G | ガラスのキズ・飛び石（ヒビは要注意） |
| Y | 亀裂・割れ（バンパー等の樹脂部品） |

> 数字が大きいほど程度が大きいことを示します。記号の付き方は会場・検査員により多少異なります。気づいたキズは記号任せにせず、状態申告でも具体的にお伝えください。

## 6. 「修復歴あり」とは？

修復歴とは、事故などで車の**骨格（フレーム）部位**を修正・交換した履歴を指します。外板（ドア・ボンネット等）の交換や板金塗装は、原則**修復歴に含まれません**。

- **修復歴に該当する骨格部位**：フレーム（サイドメンバー）／クロスメンバー／インサイドパネル／ピラー（フロント・センター・リア）／ダッシュパネル／ルーフパネル／フロアパネル（トランクフロア含む）／ラジエーターコアサポート 等の**修正・交換**
- **修復歴に当たらない例**：ドア・ボンネット・フェンダー・バンパー等の**外板パネルの交換・板金塗装**

> 修復歴があると評価点は **R / RA** となり、同条件の無事故車より査定額が下がります。「わからない・不明」も正直に申告してください。後から判明すると減額・返品・違約金の対象になります。

## 7. 査定額を左右する主な項目

| 項目 | 影響度 |
|---|---|
| 人気・相場（車種・色・ボディタイプ） | 大 |
| 年式（初度登録） | 大 |
| 走行距離 | 大 |
| 評価点・内装評価 | 大 |
| 修復歴の有無 | 大（減額） |
| グレード・主要装備 | 中 |
| 車検残・整備記録簿 | 中 |
| タイヤ・消耗品の状態 | 小〜中 |
| ボディカラー | 小〜中 |
| 出品時期・タイミング | 小〜中 |

### 走行距離の目安

一般に**年間1万km**が基準。年式に対して走行が多いと「過走行」として評価が下がり、少なすぎる場合もメーター管理の確認対象になります。

| 年式に対する走行 | 評価の傾向 |
|---|---|
| 年1万km未満（少走行） | プラス評価になりやすい |
| 年1〜1.5万km（標準） | 相場どおり |
| 年1.5万km超（過走行） | 走行が延びるほど段階的に減額 |
| 10万km超 | 大台を超えると相場が一段下がる傾向 |

> 同じ車種でも、**年式・走行・評価点**の3点で数十万円単位の差が出ます。相場は季節や需給（決算期・ボーナス期・降雪期の4WD需要 等）でも動きます。

## 8. 加点・減点ポイント

**➕ プラスに働きやすい**
- ワンオーナー・記録簿あり
- 禁煙車・内装がきれい
- 純正ナビ・純正オプション
- 人気色・定番グレード
- 車検残が長い
- タイヤ・バッテリー等が新しい

**➖ マイナスに働きやすい**
- 修復歴あり（骨格の修正・交換）
- 過走行（年式に対し走行が多い）
- 喫煙・ペット臭・内装の汚れ
- 目立つキズ・へこみ・板金塗装
- 社外・改造パーツ（車種による）
- 警告灯点灯・不具合
- 申告漏れ（後から判明）→ 減額・返品・違約金の対象

## 10. 想定手取りの考え方

```
手取り ＝ 落札額 −（出品代行手数料 ＋ 出品料 ＋ 選択オプション）
```

- **出品代行手数料**：価格帯別定額（〜50万 ¥39,800／50〜100万 ¥49,800／100〜150万 ¥59,800／150〜200万 ¥69,800／200〜300万 ¥79,800／300万超 3%）＋クラス割増
- **出品料**：¥11,000（流札は無料）
- **主なオプション**：名義変更 ¥25,000／内外装クリーニング ¥12,000〜¥30,000／代理搬入 ¥18,000 等

### 計算例：想定落札額 120万円／セダン（クリーニングなし）

| 項目 | 金額 |
|---|---|
| 落札額 | ¥1,200,000 |
| − 出品代行手数料（100〜150万円帯） | −¥59,800 |
| − 出品料 | −¥11,000 |
| **＝ 想定手取り** | **¥1,129,200** |

> 名義変更やクリーニング等のオプションを選んだ場合は、その分が差し引かれます。流札（不成約）の再出品は無料です。マイページ／`sell.html` の「出品手取りシミュレーション」で試算できます。

## 11. よくある質問（FAQ）

- **Q. 査定は無料ですか？** … 想定額の提示（机上査定）は無料。実際の出品には手数料・出品料がかかります。
- **Q. 提示額どおりに売れますか？** … 提示額は想定レンジ。最終額は当日の入札で決まり、上振れ・下振れがあります。
- **Q. 修復歴があると売れませんか？** … 売れます。無事故車より下がりますが需要はあります。正確な申告が前提。
- **Q. ローンが残っています** … 所有者がローン会社等の場合、完済・所有権解除が必要。残債・所有者情報をご連絡ください。
- **Q. 車検切れ・不動車でも可？** … 出品可能。車検整備（¥79,800・法定費用別）や搬送方法をご相談ください。
- **Q. 高く売るコツは？** … 正確な申告＋クリーニングで内装評価UP、車検残・記録簿・スペアキーを揃える、需要期に出品。
- **Q. 売れなかったら費用は？** … 流札（不成約）の再出品は無料です。

## 12. 注意点

- 提示額は想定レンジであり確定額ではありません（最終額は当日の入札で決定）。
- 評価点は会場検査員が判定するため、当社の想定と差が出る場合があります。
- 正確な状態申告が最重要。申告漏れは減額・返品・違約金の対象です。
- クレーム期間（落札日から2週間）・違約金の詳細は契約書・利用規約に準じます。水没・冠水等の重大な瑕疵は別扱い。
