# 車の査定マニュアル（AUC-AGENT／合同会社アイズ）

会員向け「車の査定マニュアル」の原稿。Webページ版は `site/appraisal.html`。

---

## 1. 査定の5ステップ

1. **情報入力** … 年式・走行・装備・状態をマイページの出品申込フォームから入力
2. **相場照合** … 直近のオートオークション（AA）成約データと照合
3. **評価点想定** … 外装・内装のコンディションを仮判定
4. **想定額提示** … 落札想定レンジを提示
5. **手取り試算** … 手数料・オプション控除後の手取りを確認

> 査定額は「確定額」ではなく、直近のAA成約実績にもとづく**想定レンジ**です。最終額は当日の入札で決まります。

## 2. 査定に必要な情報

- メーカー・車名・グレード（車検証どおり）
- 初度登録年月（年式）
- 走行距離（直近メーター値）
- 修復歴の有無（正確に申告。不明なら「不明」）
- 車検残（満了日）
- 主要装備（ナビ・サンルーフ・レザー・4WD 等）
- 外装キズ・へこみ・内装の状態
- 純正／社外パーツ・改造の有無

> 車検証・メーター・車体全体の写真があると査定精度が大きく上がります。

## 3. オートオークション 外装評価点の見方

| 評価点 | 状態の目安 |
|---|---|
| S / 6点 | 新車・未使用車クラス。ほぼ無傷。 |
| 5点 | 年式が新しく内外装とも非常に良好。目立つキズなし。 |
| 4.5点 | 走行少なめで状態良好。小キズが数か所程度。 |
| 4点 | 使用感はあるが良好。市場で最も多い一般的なコンディション。 |
| 3.5点 | キズ・へこみがやや目立つ。内装にも使用感。 |
| 3点 | キズ・へこみ・内装劣化が複数。過走行車も多い。 |
| 2点 | 外装・内装ともに難あり。要補修レベル。 |
| 1点 | 不動・全塗装・改造多数など。特殊扱い。 |
| R / RA | 修復歴あり（事故歴・骨格部の修正・交換）。減額対象。 |
| 0点 / *** | メーター交換・改ざん、水没等で通常評価不可。 |

> **4点**が「ふつうに使われてきた良好な中古車」の基準点。4点を境に上下で査定額が段階的に変わります（会場により多少差あり）。

## 4. 内装評価の見方（A〜D）

| 内装評価 | 状態の目安 |
|---|---|
| A | 非常にきれい。喫煙なし・汚れなし。 |
| B | 良好。目立つ汚れ・へたりなし。 |
| C | 使用感あり。汚れ・スレ・小さな焦げ跡など。 |
| D | 汚れ・破れ・強い臭い等が目立つ。要クリーニング。 |

> 出品前の**内外装クリーニング（サイズ別 ¥12,000〜¥30,000）**で内装評価が上がり、落札額の底上げにつながる場合があります。

## 5. 査定額を左右する主な項目

| 項目 | 影響度 |
|---|---|
| 人気・相場（車種・色・ボディタイプ） | 大 |
| 年式（初度登録） | 大 |
| 走行距離 | 大 |
| 評価点・内装評価 | 大 |
| 修復歴の有無 | 大（減額） |
| グレード・主要装備 | 中 |
| 車検残・整備記録簿 | 中 |
| タイヤ・消耗品の状態 | 小〜中 |
| ボディカラー | 小〜中 |
| 出品時期・タイミング | 小〜中 |

> 同じ車種でも、**年式・走行・評価点**の3点で数十万円単位の差が出ます。

## 6. 加点・減点ポイント

**➕ プラスに働きやすい**
- ワンオーナー・記録簿あり
- 禁煙車・内装がきれい
- 純正ナビ・純正オプション
- 人気色・定番グレード
- 車検残が長い
- タイヤ・バッテリー等が新しい

**➖ マイナスに働きやすい**
- 修復歴あり（骨格の修正・交換）
- 過走行（年式に対し走行が多い）
- 喫煙・ペット臭・内装の汚れ
- 目立つキズ・へこみ・板金塗装
- 社外・改造パーツ（車種による）
- 警告灯点灯・不具合
- 申告漏れ（後から判明）→ 減額・返品・違約金の対象

## 8. 想定手取りの考え方

```
手取り ＝ 落札額 −（出品代行手数料 ＋ 出品料 ＋ 選択オプション）
```

- **出品代行手数料**：価格帯別定額（〜50万 ¥39,800 … 300万〜 3%）＋クラス割増
- **出品料**：¥11,000（流札は無料）
- **主なオプション**：名義変更 ¥25,000／内外装クリーニング ¥12,000〜¥30,000／代理搬入 ¥18,000 等

> マイページ／`sell.html` の「出品手取りシミュレーション」で想定落札額から手取りの目安を試算できます。

## 9. 注意点

- 提示額は想定レンジであり確定額ではありません。
- 最終額は当日の入札で決定（相場変動あり）。
- 評価点は会場検査員が判定するため、当社の想定と差が出る場合があります。
- 正確な状態申告が最重要。申告漏れは減額・返品・違約金の対象です。
- クレーム期間（落札日から2週間）・違約金の詳細は契約書・利用規約に準じます。水没・冠水等の重大な瑕疵は別扱い。
