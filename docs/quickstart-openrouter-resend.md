# トライアル接続ガイド：OpenRouter（AI）＋ Resend（メール）

実費ほぼ0で「AIの応答」と「着信メール通知・一斉メール」を**本物**にするための最短手順です。
電話（Twilio）は含みません。設定はすべて **Render のバックエンドサービスの環境変数** に入れるだけです。

所要時間：約30分。クレジットカード登録は基本不要（OpenRouterの無料モデル＋Resendの無料枠）。

---

## ステップ0：どこに設定する？

1. https://dashboard.render.com を開く
2. バックエンドのサービス（Web Service）をクリック
3. 左メニュー **Environment** を開く（ここに「Key」「Value」を足していきます）
4. 値を足したら右上 **Save Changes** → 自動で再デプロイされます

> フロントではなく**バックエンド**側に入れるのがポイントです。

---

## ステップ1：OpenRouter（AIの頭脳）を接続

### 1-1. APIキーを取得
1. https://openrouter.ai にアクセスし、Googleアカウント等でサインアップ
2. 右上アイコン →「**Keys**」→「**Create Key**」
3. 名前を付けて作成 → `sk-or-v1-...` をコピー（**一度しか表示されません**）

### 1-2. Render に環境変数を追加（3つ）

| Key | Value |
|---|---|
| `OPENAI_API_KEY` | `sk-or-v1-...`（さっきのキー） |
| `OPENAI_BASE_URL` | `https://openrouter.ai/api/v1` |
| `OPENAI_MODEL` | `openai/gpt-4o-mini`（推奨）または無料モデル |

**Save Changes** を押して再デプロイ。

### モデルの選び方
- **いちばん確実で激安**：`openai/gpt-4o-mini`（日本語◎・要約も安定。月数百円規模）
- **完全無料で試す**：`meta-llama/llama-3.3-70b-instruct:free` など `:free` 付きモデル
  - 無料モデルは混雑時に遅い・JSON出力が不安定なことがあります（本システムは自動で補正しますが、要約の精度は有料モデルが上です）。

### 動作確認
管理画面 → **AI応対テスト** で話しかけて、返答が来ればAI接続成功です。
（未接続のときは定型のサンプル応答、接続後は本物のAI応答に変わります）

---

## ステップ2：Resend（メール通知）を接続

### 2-1. APIキーを取得
1. https://resend.com にサインアップ（メール認証）
2. 左メニュー **API Keys** →「**Create API Key**」→ `re_...` をコピー

### 2-2. Render に環境変数を追加（2つ）

| Key | Value |
|---|---|
| `RESEND_API_KEY` | `re_...` |
| `MAIL_FROM` | `AIオペレーター24 <onboarding@resend.dev>` |

**Save Changes** で再デプロイ。

> **ドメイン未登録でもOK**：差出人に Resend のテスト用 `onboarding@resend.dev` を使えば、
> ドメイン設定なしで送信できます。ただし**宛先は「Resendに登録したご自分のメール」だけ**に制限されます（トライアルには十分）。
> 自社ドメインから誰にでも送りたくなったら、Resendで独自ドメインを認証して `MAIL_FROM` をそのドメインのアドレスに変えます。

### 2-3. 通知先メールを自分のアドレスにする
管理画面 → **通知設定** で「通知メール」を **Resendに登録したご自分のメールアドレス** にして保存。

### 動作確認
- 管理画面 → 通話詳細 →「**通知を再送**」で、自分宛にメールが届けば成功。
- 連絡先リスト →「**一斉メール**」も同じResend経由で実送信されます（宛先がご自分のアドレスのものだけ届きます）。

## よくあるつまずき

| 症状 | 原因と対処 |
|---|---|
| AIが定型文のまま | `OPENAI_API_KEY` 未保存／再デプロイ前。Environment保存後、数分待つ |
| `401 Unauthorized` | キーの貼り間違い・余分な空白。作り直して再貼り付け |
| 要約だけ空になる | 無料モデルがJSON非対応。`openai/gpt-4o-mini` に変更で安定 |
| メールが届かない | `onboarding@resend.dev` は**自分の登録メール宛のみ**。通知先を登録メールに |
| 迷惑メールに入る | 独自ドメイン未認証だと起きやすい。本番はドメイン認証を推奨 |

## トライアル中の費用の目安

- OpenRouter：`:free` モデルなら **0円**／`gpt-4o-mini` でも軽い利用は**月数百円**
- Resend：**無料枠 100通/日・3,000通/月**。トライアルなら実質0円
- 合計：**ほぼ0円**で「AI応答＋メール通知」を本番同等に体験できます

電話を実際に鳴らす段階になったら `docs/twilio-setup.md` を参照してください（ここだけ通話実費が必要）。
