## 3. 登場人物と役割

| 登場人物 | 役割 | 主な操作 |
|---------|------|---------|
| **顧客** | 申込・進捗確認 | LINE / アプリワン |
| **本部スタッフ** | 全案件管理 | スプレッドシート / LINE WORKS。手動作業は「信販審査依頼・結果入力・加盟店選択」のみ |
| **加盟店担当者** | 商談・成約 | 加盟店専用LINE / スプレッドシート |
| **AI（GAS+OpenRouter）** | 24時間自動対応 | 自動回答・スコアリング・通知・リマインダー |

**LINE運用方針**

- 個人LINE運用 → ❌ 禁止（規約違反・自動化不能）
- LINE WORKS → 社内連絡専用
- 加盟店専用LINE公式 → 顧客対応用（採用決定）

**フランチャイズ構造**

```
本部（窓口一本化）
  ↓ 精査済み案件のみ
加盟店A・加盟店B・加盟店C...（各店専用LINE）
```

### Stage詳細

**Stage 1：ProLine自動育成（8日間ステップ）**

- ローン選択時：Step1 シミュレーター案内 / Step2 必要書類 / Step3 返済例提示 / Step4 FAQ自動回答 / Step5 申込フォーム誘導
- 買取選択時：Step6 査定の流れ / Step7 2026年中古車高値情報 / Step8 必要書類 / Step9 査定実績 / Step10 査定フォーム誘導
- → AI自動回答モードへ移行

**Stage 2：書類収集（契約者種別で分岐）**

- 個人：本人確認書類・収入証明（源泉徴収票/給与明細）・車検証（下取りあり）
- 法人：法人登記事項証明書・決算書2期分・代表者本人確認・印鑑証明書
- 個人事業主：本人確認書類・確定申告書2期分・納税証明書

**Stage 5：信販会社審査（判定別の依頼順）**

- A判定 → C001 オリコ → C002 ジャックス → C003 PG
- B判定 → C002 ジャックス → C003 PG → C004 OCS
- C判定 → C005 USS → C006 セイブサポート
- D判定 → C007 信用回復系 → C008 自社ローン

**Stage 9：成約後自動処理（7項目）**

1. 顧客へ成約完了LINE送信
2. 返済スケジュールPDF自動生成・送付
3. 返済管理シートへ自動登録
4. アフターサポート管理シートへ登録
5. アプリワンのマイページ更新
6. 本部へ成約レポート（LINE WORKS）
7. 加盟店の現在案件数を +1 更新

**Stage 10：返済・リース管理**

- 通常リマインダー：返済3日前 / 前日 / 当日
- 延滞検知：翌日フラグON → 1日目確認LINE / 5日目督促＋利息通知 / 14日超で本部・加盟店へ緊急アラート
- **延滞利息計算式**：`元金 × 年率14.6% ÷ 365 × 延滞日数`
- 自社リース：月々支払リマインダー・延滞利息自動計算・GPS有無管理

**Stage 11：アフターサポート（自動通知）**

- 車検満了90日前 / 保険満了90日前 / 乗り換え検討（完済1年前）/ 完済前90日 / 完済後フォロー

## 9. 引き継ぎ・開発時の前提

1. 全コードは GAS（JavaScript）で実装
2. **1ステップずつ**進め、実行ログ確認後に次へ
3. APIキー・トークンは Step4以降で順次入力（現状すべて空欄）
4. スプレッドシートIDは設定済み
5. **ユーザーはGAS初心者** → コードは全文提示・手順は画面操作レベル
6. 顧客接点のLINEは **既存のカーメル公式アカウント** を使用
7. 信販会社は今後8社以上に増加可能（列追加のみで対応する設計）
8. 自社リースは運用中 → 既存データに合わせ調整が必要
