### 仕組み
- **ジャンル一元管理**：`assets/js/genres.js` に1行追記すれば、LP中段カード・ジャンルハブ・ナビ・全ページのフッターチップが自動同期（外部LPtoolsページにも同JSで同期）。公開時は該当行を `url` 設定＋`status:'live'` に。
- **フォーム送信**：`assets/js/buymo.js` の `ENDPOINT` に GAS の `/exec` を設定すると、査定・問い合わせ・パートナー応募が `type:"contact"` で既存シート＋通知へ（GAS改修不要）。空欄はデモ動作。
- **ページ生成**：`tools/gen-area.js`（47県＋ハブ＋sitemap）/ `tools/gen-genre.js`（ジャンルハブ）。共通ヘッダー/フッターは `tools/_layout.js`。文言変更→再実行で全再生成。
- **技術SEO**：`sitemap.xml` / `robots.txt` / canonical（自己参照）/ JSON-LD（AutoDealer・BreadcrumbList）/ OGP。SEO構造図は `docs/buymo_seo_mindmap.(svg|png)`。

### LPtools用アセット（`site/embed/`）
- `BUYMO_共通ヘッダーフッター_LPtools.html`：外部LPに貼る共通ヘッダー/フッター（`【BASE】`を公開URLに置換）。
- ジャンル原稿9本：`LP原稿_廃車買取 / 事故車買取 / 不動車買取 / 水没車買取 / 過走行車買取 / 軽自動車買取 / トラック商用車買取 / 高級車輸入車買取 / EVハイブリッド買取.md`。

# ブラウザで http://localhost:8000/index.html を開く
