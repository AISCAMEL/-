## 4. ローカルでプレビューする
静的サイトなのでサーバー不要。簡易サーバー例：
```bash
cd site
python3 -m http.server 8000
# ブラウザで http://localhost:8000/index.html を開く
```
会員サイトは index → 無料会員登録 → mypage の順で確認（データはブラウザ内に保存）。
