## 0. 全体構成マップ

```
BUYMO（総合HP / すべて買い取る窓口・ハブ）
├─ BUYMO ECO       廃車・事故車・不動車        /eco
├─ BUYMO CLASSIC   旧車・絶版車               /classic
├─ BUYMO SUV       SUV・クロカン              /suv
├─ BUYMO CAMP      キャンピングカー            /camp
├─ BUYMO K         軽トラック（Kトラ）         /k
├─ BUYMO WHEEL     アルミホイール（部品）       /wheel
├─ BUYMO＋地域      都道府県別買取（量産テンプレ） /area/<pref>
└─ BUYMO PARTNERS  FC加盟店募集               /fc
```

- **メインHP**：総合窓口。あらゆる車・パーツを買い取るイメージ。各専門LPへの導線（ハブ）。
- **専門特化LP**：広告流入・専門ニーズ用。1カテゴリ＝1LPで訴求を尖らせる。
- **地域別LP**：1テンプレを地域名差し替えで量産（SEO/MEO）。
- **FC募集**：加盟店募集。

## 1. 共通LPテンプレート（全LP共通の骨格）

専門LP・地域LPはこの骨格を踏襲し、訴求コピーと実績のみ差し替える。

| # | セクション | 目的 | 主な要素 |
|---|-----------|------|---------|
| 1 | ファーストビュー（FV） | 3秒で「ここで売れる」を伝える | キャッチコピー / 対象車種ビジュアル / 一括査定CTA / 信頼バッジ（実績件数・年中無休・無料） |
| 2 | 簡易査定フォーム（FV内 or 直下） | 離脱前に最短入力 | メーカー・車種・年式・走行距離・連絡先（後述§3） |
| 3 | 選ばれる理由（強み3〜5） | 差別化 | 高価買取/手数料0円/出張・引取無料/スピード入金/廃車手続き代行 等 |
| 4 | 買取実績・査定例 | 価格の納得感 | 車種別の買取価格例・お客様の声 |
| 5 | 買取の流れ（3〜4ステップ） | 不安解消 | 申込→査定→契約→入金 |
| 6 | よくある質問（FAQ） | 反論処理 | 不動車OK？/書類は？/手数料は？/即日可？ |
| 7 | 対応エリア | 地域訴求 | 全国対応／出張買取エリア |
| 8 | 最終CTA + フォーム再掲 | 申込確定 | 電話・LINE・フォーム |
| 9 | 運営情報・フッター | 信頼 | 会社情報・古物商許可番号・プライバシーポリシー |

**共通CTA**：①無料査定フォーム ②LINE査定（BUYMO公式LINE）③電話。
**共通トラッキング**：LP別の流入計測（広告用UTM）。送信データは §開発仕様書のスコアリング/案件管理へ連携。

## 2. LP別の訴求ポイント

### BUYMO ECO（廃車・事故車・不動車） `/eco`
- **訴求**：動かない・事故・水没でも0円以上／廃車費用0円／**廃車手続き・名義変更を無料代行**／還付金（自動車税・自賠責・重量税）が戻る。
- **キャッチ例**：「動かない車も、1円でも高く。廃車費用0円＋還付金もお返し。」
- **固有フォーム項目**：走行可否（自走/不動/事故）、車検残、書類の有無。

### BUYMO CLASSIC（旧車・絶版車） `/classic`
- **訴求**：年式が古いほど価値が出る場合あり／旧車・絶版車・レストアベースの専門査定／不動でもOK。
- **キャッチ例**：「その旧車、価値を知る専門店へ。絶版車・レストアベース高価買取。」
- **固有フォーム項目**：年式（〜1990等の選択肢を厚く）、改造・オリジナル状態、不動可否。

### BUYMO SUV（SUV・クロカン） `/suv`
- **訴求**：SUV・クロカン人気で高値継続／4WD・社外パーツ評価／過走行でも実績多数。
- **キャッチ例**：「SUV高騰中。今が売り時。4WD・社外カスタムも高評価。」
- **固有フォーム項目**：駆動方式（2WD/4WD）、社外パーツ有無。

### BUYMO CAMP（キャンピングカー） `/camp`
- **訴求**：キャンピングカー・バンコン専門査定／架装・FFヒーター等の装備を加点／全国出張査定。
- **キャッチ例**：「キャンピングカー専門査定。架装・装備までしっかり評価。」
- **固有フォーム項目**：ベース車両、架装ビルダー、就寝人数、装備（電装/トイレ/ヒーター）。

### BUYMO K（軽トラック / Kトラ） `/k`
- **訴求**：軽トラは過走行・多少の傷でも需要大／農機・業務利用車も歓迎／海外需要で高値。
- **キャッチ例**：「軽トラ、まだ高く売れます。過走行・キズありOK。」
- **固有フォーム項目**：荷台状態、4WD/エアコン/パワステ有無。

### BUYMO WHEEL（アルミホイール・部品） `/wheel`
- **訴求**：車両なしでもホイール単体買取／社外・純正OP・希少サイズ歓迎／タイヤ付きも可。
- **キャッチ例**：「ホイールだけでも買取。社外・純正・希少サイズ歓迎。」
- **固有フォーム項目**：メーカー/銘柄、インチ・PCD・本数、タイヤ有無、ガリ傷状態。※車両フォームと別系統（部品買取フロー）。

### BUYMO＋地域（都道府県別） `/area/<pref>`
- **訴求**：「<地域>で車を高く売るなら」地域密着・出張買取・即日対応。
- **テンプレ量産方針**：1テンプレに対し ①地域名 ②対応エリア（市区町村）③地域実績/お客様の声 を差し替え。47都道府県 or 主要エリア展開（方針は要確定）。
- **SEO/MEO**：title/h1/メタに「<地域>＋車買取」、Googleビジネスプロフィール連携。

## 3. 買取フォーム項目（標準セット）

メインHP・各車両LP共通の標準フォーム。`BUYMO WHEEL` のみ部品用に差し替え。

**車両買取フォーム（標準）**

| 区分 | 項目 | 必須 | 備考 |
|------|------|------|------|
| 車両 | メーカー | ● | プルダウン |
| 車両 | 車種・モデル | ● | |
| 車両 | 年式 | ● | |
| 車両 | 走行距離 | ● | 区分選択でも可 |
| 車両 | グレード/型式 | － | |
| 車両 | 車検残 | － | |
| 状態 | 走行可否（自走/不動/事故） | ●（ECO等） | LP別に出し分け |
| 状態 | 傷・修復歴 | － | |
| 連絡 | 氏名 | ● | |
| 連絡 | 電話番号 | ● | |
| 連絡 | メール or LINE | ● | いずれか |
| 連絡 | 都道府県 | ● | 出張査定の振り分け |
| 同意 | 個人情報取扱い同意 | ● | チェック |

**部品買取フォーム（BUYMO WHEEL）**：銘柄／インチ／PCD／本数／タイヤ有無／状態（ガリ傷）／写真アップロード／連絡先／同意。

- **連携**：送信→開発仕様書 §4 Stage2-4（書類収集・AIスコアリング）／買取案件管理シートへ登録。
- **写真アップロード**：状態確認の精度向上のため任意で添付可（査定スピードUP）。
