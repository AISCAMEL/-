## 1. 利用規約（会員サイト共通）

第1条（適用）本規約は、`[事業者名]`（以下「当社」）が提供するオークション代行サービス（購入代行・出品代行、以下「本サービス」）の利用に関し、会員と当社の間に適用される。

第2条（会員登録）利用希望者は当社所定の方法で登録を行い、当社が承認した時点で会員資格が生じる。登録情報は正確かつ最新に保つものとする。

第3条（本サービスの内容）本サービスは、業者オートオークションにおける車両の入札・落札の代行（購入代行）、および車両の出品・売却の代行（出品代行）を行う。当社は古物営業法に基づく許可（許可番号：`[　]`）のもとに営業する。

第4条（手数料）会員は、当社所定の代行手数料・諸費用を支払う。料金は本サービスサイトに掲示する料金表による。

第5条（禁止事項）法令違反、虚偽申告（車両状態・本人確認等）、第三者へのなりすまし、当社業務の妨害等を禁止する。

第6条（反社会的勢力の排除）会員は、自らが反社会的勢力でないことを表明・確約する。

第7条（免責）当社は、オークション相場の変動、落札の成否、会場・第三者の事由による損害について、当社の故意・重過失による場合を除き責任を負わない。

第8条（個人情報）当社はプライバシーポリシーに従い個人情報を取り扱う。

第9条（規約変更）当社は必要に応じ本規約を変更でき、サイト掲示をもって効力を生じる。

第10条（準拠法・合意管轄）本規約は日本法に準拠し、紛争は `[　]` 裁判所を第一審の専属的合意管轄裁判所とする。

## 2. 購入代行契約書（雛形）

**自動車オークション購入代行契約書**

委託者（会員）`[氏名]`（以下「甲」）と、受託者 `[事業者名]`（以下「乙」）は、次のとおり契約する。

- 第1条（委託）甲は乙に対し、業者オークションにおける車両の入札・落札の代行を委託する。
- 第2条（対象・上限）対象車両の条件および入札上限価格は、申込内容（マイページ記録）のとおりとする。乙は上限価格を超えて入札しない。
- 第3条（手数料・費用）甲は乙に対し、料金表に定める代行手数料、落札料、リサイクル料等の実費、陸送費、選択オプション費用を支払う。
- 第4条（支払）甲は、落札成立後 `[3]` 営業日以内に総額を乙の指定口座へ支払う。期限超過時は年 `[14.6]`% の遅延損害金を申し受ける。
- 第5条（落札後の辞退・違約金）落札成立後に甲が辞退した場合、甲は違約金として `[代行手数料相当額＋落札価格の10%]` を支払う。
- 第6条（車両の状態）乙はオークション検査票・評価点に基づき確認するが、会場の評価を超える保証はしない。クレームは第7条による。
- 第7条（クレーム期間）落札車両の状態相違については、落札日から **2週間以内**（水没等の重大な瑕疵は別扱い）を受付期間とし、会場規定に従い乙が窓口として対応する。
- 第8条（引渡し）乙は陸送・名義変更等を行い、甲の指定場所へ車両を引き渡す。
- 第9条（免責・不可抗力）天災・会場事由・第三者事由等による不成立・遅延について、乙は故意重過失を除き責任を負わない。
- 第10条（合意管轄）本契約の紛争は `[　]` 裁判所を専属的合意管轄とする。

## 3. 出品代行契約書（雛形）

**自動車オークション出品代行契約書**

委託者（会員）`[氏名]`（以下「甲」）と、受託者 `[事業者名]`（以下「乙」）は、次のとおり契約する。

- 第1条（委託）甲は乙に対し、甲所有車両の業者オークションへの出品・売却の代行を委託する。
- 第2条（車両情報の申告）甲は車両情報（走行距離・修復歴・装備・状態）を正確に申告する。乙は当該情報をもとに**デジタル出品票**を作成し甲へ郵送する。
- 第3条（搬入）甲は、乙の指定する会場へ、開催日前日までに車両・必要書類を搬入する（代理搬入オプションを除く）。
- 第4条（手数料・費用）甲は乙に対し、料金表に定める出品代行手数料（落札価格の `[3]`%、最低 `[33,000]` 円）、成約料、選択オプション費用を支払う。これらは成約代金から控除できる。
- 第5条（入金）乙は、成約・入金確認後 `[5]` 営業日以内に、手数料等を控除した金額を甲の指定口座へ支払う。
- 第6条（申告不一致・減額・違約金）甲の申告と車両の実状態が異なり、減額・返品・落札者クレームが生じた場合、甲はこれにより生じた費用・違約金を負担する。出品後の甲都合による取下げは違約金 `[33,000円＋実費]` とする。
- 第7条（クレーム期間）落札後のクレーム（落札日から **2週間以内**／水没等の重大な瑕疵は別扱い・会場規定による）について、乙が窓口として対応する。
- 第8条（所有権・名義）売却に伴う名義変更等の手続は乙が代行する。残債がある場合は甲が事前に通知する。
- 第9条（免責）相場変動・流札・会場事由による不成立について、乙は故意重過失を除き責任を負わない。
- 第10条（合意管轄）本契約の紛争は `[　]` 裁判所を専属的合意管轄とする。

## 4. 違約金・クレーム期間 規定（要約掲示用）

- **クレーム期間**：落札日から **2週間以内**。水没・冠水等の重大な瑕疵は別扱い。対象は「申告と異なる重大な瑕疵・メーター不正・主要機関の重大不具合」等、会場規定に準じる。
- **購入・落札後辞退の違約金**：`[代行手数料相当額＋落札価格の10%]`。
- **出品・申告不一致／取下げ**：減額・返品費用の負担、または `[33,000円＋実費]`。
- **支払遅延**：年 `[14.6]`% の遅延損害金。
