# 🌊 IWASAWA SURF BASE — Supabase 接続ガイド（最短3ステップ）

このガイドの通りにやれば、モックで見た画面が「本物のデータで動くアプリ」になります。
所要時間の目安：**15〜20分**。むずかしいコードは書きません。

---

## 事前に必要なもの
- メールアドレス（Supabaseの無料アカウント用）
- このプロジェクト（`web/` フォルダ）が手元にある状態
- パソコンにNode.js（v18以上）が入っていること

---

## ステップ1：Supabaseプロジェクトを作る（あなたの作業）

1. https://supabase.com を開き「Start your project」→ GitHub かメールで**サインアップ**
2. 「New project」をクリック
   - Name：`iwasawa-surf-base`（何でもOK）
   - Database Password：好きなパスワード（**メモしておく**）
   - Region：`Northeast Asia (Tokyo)` を推奨
3. 「Create new project」→ 1〜2分で準備完了

## ステップ2：テーブルを作る（コピペ1回）

1. 左メニューの **SQL Editor** を開く
2. `web/supabase/schema.sql` を**全部コピー**して貼り付け
3. 右下の **Run** を押す（"Success" と出ればOK）
   - → 会員・投稿・スキル・通報などのテーブルが一気に作られます
4. 左メニュー **Authentication → Providers → Email** が有効になっているか確認
   - 開発中は同じ画面の「Confirm email」を**オフ**にすると、登録がすぐ通って楽です

## ステップ3：鍵を設定してアプリを起動（コピペ2回）

1. 左メニュー **Project Settings → API** を開き、次の2つをコピー：
   - `Project URL`
   - `anon public` キー
2. `web/` フォルダで、`.env.example` を `.env.local` という名前でコピーし、値を貼り付け：
   ```
   NEXT_PUBLIC_SUPABASE_URL=（Project URL を貼る）
   NEXT_PUBLIC_SUPABASE_ANON_KEY=（anon public を貼る）
   NEXT_PUBLIC_SITE_URL=http://localhost:3200
   ```
3. ターミナルで `web/` に移動して起動：
   ```bash
   npm install
   npm run dev
   ```
4. ブラウザで **http://localhost:3200** を開く 🎉
   - ※ このアプリは **3200番ポート**で起動します（3000番は別アプリ用）
   - ※ 別のポートにしたい場合は `package.json` の `-p 3200` の数字を変えてください

---

## 動作チェック（この順でどうぞ）

1. `/signup` で新規登録 → 自動で Beginner / Free になる
2. `/community` を開く → `/community/new` で投稿してみる → いいね・コメント
3. `/skills` → 「出品する」→ 別アカウントで申し込んでみる
4. `/waves` → 波情報が表示される
   - ※ 波データが出ない場合、社内ネットワーク等で `api.open-meteo.com` /
     `marine-api.open-meteo.com` への通信が塞がれている可能性。許可すると出ます。

## 自分を「運営者」にする（管理画面を見るため）

1. Supabase の **Table Editor → members** を開く
2. 自分の行の `role` を `admin` に変更して保存
3. アプリで `/admin` を開く → ダッシュボード・投稿管理・会員管理・通報が使えます
4. 以降は `/admin/members` から他の会員の種別（local昇格など）を変更できます

---

## つまずいたら
- 画面に「読み込みに失敗」→ `.env.local` の値が正しいか、`npm run dev` を再起動
- 登録後にメールが来ない → Authentication で「Confirm email」をオフに
- それでもダメなら、エラーメッセージをそのまま教えてください。一緒に直します。
