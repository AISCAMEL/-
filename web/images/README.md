# 画像の入れ方（BUYMOメインLP）

このフォルダ `web/images/` に、下記の**ファイル名どおり**に画像を置くと、LP（`web/index.html`）へ自動で表示されます。
画像がまだ無い箇所は「写真＋ファイル名」のガイドが表示され、レイアウトは崩れません。

## 用意する画像一覧

| ファイル名 | 使う場所 | 推奨サイズ（目安） | 備考 |
|-----------|---------|------------------|------|
| `hero.jpg` | ファーストビューの全画面背景 | 1600×900px（横長） | 車・スタッフ等。文字が乗るので大きめ・やや暗めが◎。置くと自動で全画面写真に切替 |
| `band-1.jpg` | 全幅ビジュアル帯①「動かない車も〜」 | 1600×900px | 暗めの写真が映える（任意） |
| `band-2.jpg` | 全幅ビジュアル帯②「下取りより高い〜」 | 1600×900px | 暗めの写真が映える（任意） |
| `result-1.jpg` | 買取実績①（アクア） | 600×400px | 車両写真 |
| `result-2.jpg` | 買取実績②（ヴェルファイア） | 600×400px | 車両写真 |
| `result-3.jpg` | 買取実績③（マーチ） | 600×400px | 車両写真 |
| `result-4.jpg` | 買取実績④ | 600×400px | 車両写真 |
| `voice-1.jpg` | お客様の声① | 200×200px（正方形） | 顔/車/アイコン。丸く表示 |
| `voice-2.jpg` | お客様の声② | 200×200px（正方形） | 同上 |
| `voice-3.jpg` | お客様の声③ | 200×200px（正方形） | 同上 |

## 入れ方の手順

1. 画像を上の表の**ファイル名に変更**する（例：`愛車.jpg` → `result-1.jpg`）。
2. この `web/images/` フォルダに置く（アップロードする）。
3. `web/index.html` をブラウザで開くと反映されます。

## 注意
- 形式は `.jpg` 前提です（`.png` を使う場合は `index.html` の拡張子も合わせて変更が必要 → ご相談ください）。
- 画像は軽く（1枚あたり目安300KB以下）すると表示が速くなります。
- 画像を差し替えたい・追加で写真枠を増やしたい場合は教えてください（店舗写真・スタッフ紹介など）。

## キャラクター素材（characters/）

メインマスコットは **ラクダ（CarMel＝Camel）**。PDF/AIから透過PNGを生成：
- `camel-face.png` … ラクダの顔（透過・約900px）
- `camel-face-flip.png` … 左右反転版
- `camel-icon.png` … ヘッダー/バッジ用の小サイズ
LPでの使用：ヘッダーロゴ横／ヒーロー／査定フォーム上の覗き／STEP案内／最終CTA。
※収録ポーズは「顔」1種のため、サイズ・向きで変化を付けて配置。全身・車に乗ったポーズ等の
　透過PNGがあれば、さらに多彩に配置できます（いただければ追加します）。
（参考：frog-*.jpg は別キャラのため未使用。）
