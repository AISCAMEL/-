=== Carmel Chat Widget ===

カーメル相談AIのチャットウィジェットを、WordPressサイトの右下に表示するプラグインです。

== インストール ==
1. WordPress管理画面 →「プラグイン」→「新規追加」→「プラグインのアップロード」
2. この carmel-chat.zip を選んでアップロード →「今すぐインストール」
3. 「プラグインを有効化」

== 設定 ==
1. 管理画面 →「設定」→「Carmel Chat」を開く
2. 「接続先ホスト」に、チャットの頭脳(Nodeアプリ)を公開したURLを入力
   例: https://chat.example.com
3. 必要ならLINE URL・電話番号も入力 →「保存する」

保存すると、サイトの全ページ右下にチャットアイコンが表示されます。

== 注意 ==
- 「接続先ホスト」は HTTPS で公開されている必要があります。
- 接続先(Nodeアプリ)側で、環境変数 ALLOWED_ORIGINS にこのWordPressのドメインを設定してください。
- 接続先が未設定の間は、チャットは表示されません（サイトに影響しません）。
