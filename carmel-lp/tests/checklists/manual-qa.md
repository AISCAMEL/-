## 機能テスト
- [ ] F-01 初回表示でウィジェットが右下に常時表示される
- [ ] F-02 ロード3秒後に自動ポップアップが1回だけ出る
- [ ] F-03 自動ポップアップを閉じると同セッションで再表示しない
- [ ] F-04 ウィジェット押下で手動ポップアップ（AIチャット）が開く
- [ ] F-05 ×で閉じるとウィジェットのみ残る
- [ ] F-06 外側クリックで閉じる
- [ ] F-07 Escキーで閉じる（PC）
- [ ] F-08 LINE CTAが指定URLへ遷移
- [ ] F-09 電話CTAが tel リンク起動

## AIチャットボット
- [ ] C-01 ポップアップ初回表示で挨拶＋サジェストが出る
- [ ] C-02 メッセージ送信でユーザー吹き出し→タイピング→AI応答が流れる
- [ ] C-03 応答がストリーミング表示される
- [ ] C-04 サジェストチップのタップで質問が送信される
- [ ] C-05 通信失敗/キー未設定時にフォールバック文言＋LINE/電話導線が出る
- [ ] C-06 審査合否を断定しない／個人情報を聞き出さない応答である
- [ ] C-07 Enterで送信・Shift+Enterで改行
- [ ] C-08 連投・空送信で多重発火や例外が起きない
