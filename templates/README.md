# 📂 テンプレート一式（請求書 → マネーフォワード 本線）

このフォルダには、Gmail請求書の自動仕分け → マネーフォワード連携の**本線**で使うスプレッドシート雛形（CSV）を入れています。
**まずはここを埋めることが、この案件で一番重要な「業務判断」**です（GASのコードより先に効きます）。

## ファイル一覧

| ファイル | 役割 | 誰が埋める |
|----------|------|-----------|
| `業者マスタ.csv` | 業者ごとの「既定の勘定科目・税区分・源泉有無・送信元アドレス」 | **経理が埋める（最重要）** |
| `仕訳パターン明細.csv` | 仕訳の組み立て方（借方/貸方の作り方）の定義 | 初期は同梱の雛形のまま |
| `勘定科目マスタ.csv` | 使ってよい勘定科目・税区分の一覧（入力チェック用） | MFの設定に合わせて調整 |
| `請求台帳.csv` | 請求一覧（GASが自動追記する）の列定義＋見本1行 | 自動（人は確認・修正のみ） |
| `処理ログ.csv` | 監査ログの列定義 | 自動 |

## Googleスプレッドシートへの取り込み方

1. Googleドライブで新規スプレッドシートを1つ作成（名前例：`請求書管理`）。
2. シートタブを5枚用意し、それぞれ `業者マスタ` `仕訳パターン明細` `勘定科目マスタ` `請求台帳` `処理ログ` と命名。
3. 各CSVを「ファイル → インポート → このシートを置換」で対応タブに読み込む。
4. このスプレッドシートのIDを、後でGASのスクリプトプロパティに登録します。

## 使い方のキモ

- **`業者マスタ` を1業者ずつ埋めるだけで、その業者の仕訳が自動で決まります。**
  - 「送信元アドレス」を入れると、メールの送り主から業者を自動特定できます（精度が大きく上がります）。
  - 「既定借方勘定科目」「既定税区分」を入れると、その業者の仕訳科目が自動で入ります。
  - 「仕訳パターンID」で、源泉あり/なし等の仕訳の型を選びます。
- `仕訳パターン明細` の `{業者.既定借方勘定科目}` のような **波かっこ `{}` は「あとで実際の値に置き換わる場所」**を表します（プレースホルダといいます）。
- `自動承認可否` を `否` にしておくと、その業者は当面ずっと人の確認に回ります（新規業者の慣らし運転に使用）。

## 金額の入れ方（税抜入力 / 税込入力）

`仕訳パターン明細.csv` は **税抜入力（仮払消費税を分けて計上）** を初期値にしています。
マネーフォワード側を税込入力で運用している場合は、確認事項#5の回答に合わせてパターンを差し替えます。

## マネーフォワードCSV形式について（重要）

最終的にMFへ取り込む「仕訳インポートCSV」の**正確な列名・順序・税区分の表記は、MFの製品とバージョンで異なります**。
実装前に、**あなたのMF管理画面から「仕訳インポート用テンプレートCSV」を1つダウンロード**してください。その実物に出力を合わせ込みます。
（このフォルダの雛形は、MF取込の「手前」＝社内台帳としての完成形です。）
