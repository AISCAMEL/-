# 運用マニュアル｜AIS Corporate テーマ

合同会社アイズ コーポレートサイトの**運用・編集ガイド**です。
「どこを触ると、どこが変わるか」を中心に、管理画面での操作と、テーマファイルの編集箇所をまとめています。

> 技術的な導入・ビルド手順は `README.md` を参照してください。

---

## 0. 編集の全体像

このサイトのコンテンツは、大きく2か所にあります。

| 種類 | 編集場所 | 例 |
|---|---|---|
| 記事・ブログ的なもの | **WordPress 管理画面** | お知らせの追加・編集 |
| サイトの構成・定型コンテンツ | **テーマのデータ層**（`inc/data.php`） | 事業内容・実績・募集職種・理念・会社情報 |

> `inc/data.php` は「サイト全体の文章をまとめた台本」です。ここを直すと該当ページに即反映されます。
> 編集時は**全角・半角の引用符**や**カンマ**を崩さないよう注意してください（PHPの配列です）。不安な場合はバックアップを取ってから編集を。

## 1. 管理画面でできること

### 1-1. お知らせ・コラムの追加
1. 管理画面 → **お知らせ・コラム** → 新規追加
2. タイトル・本文を入力 → 公開
3. 一覧（`/news`）と詳細ページに自動で反映されます。

> 本文を入力した記事は、その本文が優先表示されます。未入力の場合はテーマ内のサンプル文が表示されます。

### 1-2. 固定ページの基本情報
会社概要・理念・FAQ などの**ページ自体**は、有効化時に自動生成されています
（管理画面 → 固定ページ で確認できます）。各ページの本文デザインはテンプレートで作っているため、
文言の多くは後述の `inc/data.php` で管理しています。

### 1-3. お問い合わせの通知先
お問い合わせフォームの送信先は、**管理画面 → 設定 → 一般 → 管理者メールアドレス**です。
ここを実際の受信用アドレスに設定してください。

## 2. テーマのデータ層（`inc/data.php`）の編集

ファイル：`wp-content/themes/ais-corporate/inc/data.php`
各項目は「関数」で分かれています。該当の関数を探して、文字列を書き換えるだけです。

### 2-1. 会社の基本情報　`ais_site()`
社名・タグライン・メール・住所・返信目安など。
```php
'email' => 'info@aisjaltd.com',   // ← ここを変更
'address' => '〒979-0204 福島県…',
```

### 2-2. 会社概要の表　`ais_company_profile()`
会社概要ページの「会社名・代表者・所在地…」の表。

### 2-3. 事業・ブランド　`ais_services()`
カーメル／BUYMO／CARSHICO／天護／レッカー／FC／APPREX／WEB crews／AIオペレーター24。
各事業の `name`（事業名）`brand`（ブランド名）`tagline`（一言）`summary`（説明）
`highlights`（特長）`offerings`（メニュー）`external_url`（公式サイト）などを編集できます。
ここを直すと、トップ・事業紹介・各詳細・ブランド一覧・FC・チャット回答まで反映されます。

### 2-4. 実績　`ais_works()` ／ お知らせ（サンプル）　`ais_news()`
構成確認用のサンプルです。実データに差し替えたら、各項目の `is_placeholder` を `false` にすると
「サンプル」表示が消えます。お知らせは管理画面での追加も可能（1-1参照）。

### 2-5. 募集職種　`ais_recruit_positions()`
採用ページの職種カード。`title`（職種名）`type`（雇用形態）`brand`（取扱ブランド）
`summary`（業務内容）`points`（箇条書き）`welcome`（歓迎する方）。
職種の追加・削除は、配列のブロックを丸ごと足す／消すだけです。

### 2-6. 理念　`ais_philosophy()`
パーパス・ミッション・ビジョン・私たちの想い・バリュー（革新／品質／信頼性）。

### 2-7. 代表メッセージ　`ais_representative()`
氏名・肩書き・本文（段落の配列）。

### 2-8. ナビゲーション　`ais_main_nav()` ／ `ais_footer_nav()`
ヘッダーのメニュー（ドロップダウン含む）とフッターのリンク列。

### 2-9. トップの見出し（キャッチコピー）
ファイル：`template-parts/home/hero.php`
```php
クルマも、デジタルも。…「困った」を、まるごと。   // ← ここがメイン見出し
```

## 3. AIチャット（コンシェルジュ）

### 3-1. 動かすための設定
1. [openrouter.ai](https://openrouter.ai/) で API キーを発行
2. キーを設定（どちらか）
   - **推奨**：`wp-config.php` に `define('AIS_OPENROUTER_API_KEY', 'sk-or-...');`
   - 簡単：管理画面 → **設定 → AIチャット** に入力
3. 同画面で **有効化／モデル／あいさつ文／追加指示** を調整

### 3-2. 回答内容の調整
チャットは `inc/data.php` の会社情報・事業・FAQ を自動で読み込みます。
**事業内容を更新すれば、AIの回答も自動で追従**します。
特定の案内を足したいときは、設定画面の「追加のAI指示」に書いてください。

### 3-3. 注意
- 電話番号は案内しない方針が組み込まれています。
- 不確実な内容はお問い合わせフォームへ誘導します。

## 4. オペレーター映像・アバターの差し替え

`assets/media/` の以下3ファイルを置き換えるだけで、案内係（人物）を変更できます。

| ファイル | 用途 | 目安 |
|---|---|---|
| `operator.mp4` | ループ映像（ミュート） | 480×270 / H.264 / 数秒〜十数秒 |
| `operator-poster.jpg` | 映像の静止画 | 480×270 |
| `operator-avatar.jpg` | 丸アイコン | 正方形 256px |

> `operator-avatar.jpg` が無い場合は内蔵イラストに自動で切り替わります。
> **人物の写真・映像は、必ず利用許諾（本人同意・素材ライセンス）をご確認ください。**

---

## 5. ロゴ・favicon の差し替え

- **ロゴ**：`assets/img/` に配置し、`header.php` ／ `footer.php` のロゴ部分で参照します。
- **favicon／アプリアイコン**：`assets/img/` の `favicon-32.png` / `apple-touch-icon.png` /
  `favicon.svg` を差し替え（`functions.php` の `ais_favicon()` が出力）。
  管理画面 → 外観 → カスタマイズ → サイト基本情報 で「サイトアイコン」を設定した場合は、そちらが優先されます。

---

## 6. デザイン（Tailwind）を編集したとき

クラス（見た目）を変更した場合は、CSS の再ビルドが必要です。
```bash
npx tailwindcss -c ./tailwind.config.js -i ./assets/css/src.css -o ./assets/css/theme.css --minify
```
文章だけの編集（`inc/data.php` 等）では再ビルドは不要です。

## 8. 困ったとき（FAQ）

| 症状 | 確認ポイント |
|---|---|
| チャットが表示されない | 有効化されているか／OpenRouter キーが設定されているか（設定→AIチャット） |
| ページがリンク切れ（404） | 設定→パーマリンクを一度「保存」したか |
| 音声が鳴らない | チャットヘッダーのスピーカーがミュートでないか／OSに日本語音声があるか／実ブラウザで操作したか |
| 文章を直したのに変わらない | `inc/data.php` の正しい関数を編集したか／キャッシュを再読み込み |
| 見た目が崩れた | Tailwind の再ビルドが必要な変更をしていないか（第6章） |
