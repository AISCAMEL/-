# クラウド会計（自社版）

freee のような会計ソフトを **自社開発** した最小構成版（MVP）です。
サーバー不要・**ブラウザ単体で動作**し、データはブラウザ内（IndexedDB）に保存されます。

## 使い方

`accounting/index.html` をブラウザで開くだけです。ビルドやインストールは不要です。

- ローカルで開く場合：`index.html` をダブルクリック
- サーバーに置く場合：`accounting/` フォルダをそのままアップロード（静的ホスティングでOK）

> データはお使いのブラウザ内にのみ保存されます。別のPC・ブラウザとは共有されません。
> 共有・引っ越し・バックアップは、**設定 → バックアップ**の書き出し/取り込み（JSON）で行います。

## 主な機能

| 区分 | 機能 |
|------|------|

| レポート | **総勘定元帳**、**合計残高試算表**、**決算書（貸借対照表 BS / 損益計算書 PL）**、**部門別損益**、**予算・予実管理**、**消費税集計**（インボイス対応・原則/簡易課税/2割特例・税率別・納付試算）、**期首残高・繰越**（前年度末から自動繰越）、**決算・確定申告**（損益振替・決算報告書）、**申告サポート**（所得金額調整・法人税概算・CSV出力）、**電子申告データ**（消費税申告書サマリ・法人税基礎データ・財務諸表/内訳CSV）、**訂正・削除履歴**（電帳法） |

### 追加機能の使い方

- **銀行明細取込**：銀行のCSVを選び、列（日付・摘要・出金/入金）を対応づけて一括で仕訳に計上します。摘要のキーワードから相手科目を推定する「自動仕訳ルール」を登録できます（文字コードは UTF-8 / Shift_JIS に対応）。
- **固定資産・減価償却**：資産を登録すると当期の減価償却を自動計算し、ワンクリックで仕訳計上できます。償却方法は**定額法／定率法（200%）／一括償却資産（3年均等）／少額減価償却資産（即時償却）**に対応。初年度は月割で計算します。
- **期首残高・繰越**：他システムからの移行時の期首残高入力に加え、「前年度末から繰越」で前期のBS残高＋当期純利益（繰越利益剰余金へ振替）を自動反映します。
- **決算・確定申告**：年度締めチェックリスト、**損益振替（決算整理仕訳）**の自動生成、会社情報・BS・PL・消費税・固定資産をまとめた**決算報告書**の印刷/PDF出力。損益振替仕訳は繰越利益剰余金に反映しつつ、損益計算書の表示には当期の実績を残します。
- **証憑（電帳法）**：領収書・請求書の画像/PDFを保存し、**電子帳簿保存法の検索要件（取引年月日・金額・取引先）**で検索できます。仕訳入力画面から直接添付も可能です。
- **消費税の計算方式**：消費税集計ページで**原則課税／簡易課税（第1〜6種のみなし仕入率）／2割特例**を切り替えて納付額を試算・比較できます。
- **申告サポート**：会計上の当期純利益に税務上の加算・減算を加減して**所得金額**を計算（別表四の考え方）、法人税等の概算を表示。**勘定科目内訳（試算表）・仕訳帳のCSV出力**（Excel対応・BOM付きUTF-8）で税理士への共有も簡単です。
- **訂正・削除履歴**：仕訳・請求書・固定資産・証憑・給与の作成／訂正／削除を自動記録します（電帳法の訂正・削除履歴に対応、追記のみで改変不可）。
- **部門別損益**：部門マスタを登録し、仕訳に部門を付けると、部門ごとの売上・費用・利益を集計できます（部門未指定は「未配賦」）。
- **給与計算**：従業員マスタを登録し、基本給・手当から社会保険料（健康・厚生年金・雇用／料率設定可）と**源泉所得税（電子計算機計算の特例・甲欄／扶養人数対応）**を控除して手取を算出。**給与仕訳（借：給料手当／貸：預り金・普通預金）を自動計上**します。**年末調整**では年間の給与・社保・源泉を集計して年税額と過不足（還付／追徴）を計算し、仕訳計上できます。
- **予算・予実管理**：科目ごとに年間予算を入力し、当期実績との差異・達成率、予算利益と実績利益の対比を確認できます。
- **e-Tax取込用データ**：財務諸表（決算書）・売掛金内訳のCSVを出力し、e-Tax の財務諸表・勘定科目内訳明細書の作成に取り込む元データとして利用できます。
- **電子申告データ**：消費税申告書の各欄（課税標準額・国税/地方の消費税額・控除税額・納付税額）を割戻し計算し、法人税の基礎データとあわせて画面表示・CSV出力します。e-Tax/eLTAX へ転記・取込する際の基礎データとしてご利用ください。
- **クラウド同期**：同梱の同期サーバー（`server/`）を起動し、同じワークスペース名を設定すると、複数の担当者・端末で同じ帳簿を共有できます。詳細は [`server/README.md`](server/README.md)。

### 会計仕様

- **複式簿記**：借方＝貸方の一致を入力時にチェック。5要素（資産・負債・純資産・収益・費用）で自動分類。
- **消費税**：税込経理方式。10% / 8%（軽減税率）の税区分に対応。税率区分ごとに端数処理。
- **インボイス**：会社の適格請求書発行事業者 登録番号（T＋13桁）を請求書に表示。税率別の対象額・消費税を明記。
- **会計年度**：期首月を設定可能（初期値 4月始まり）。各レポートは期間で絞り込み可能。

## 技術構成

- 純粋な HTML / CSS / JavaScript（ビルド不要・外部依存なし・ES モジュール不使用）
- 永続化：IndexedDB（`assets/js/db.js`）
- 画面遷移：ハッシュルーター（`assets/js/ui.js`）

```
accounting/
├─ index.html
└─ assets/
   ├─ css/app.css
   └─ js/
      ├─ util.js        金額・日付・DOM ユーティリティ
      ├─ accounts.js    勘定科目マスタ・税区分の定義
      ├─ db.js          IndexedDB・初期データ・バックアップ
      ├─ store.js       ドメイン層（設定/科目/取引先/仕訳/請求書）
      ├─ reports.js     帳票計算（元帳・試算表・BS/PL・消費税）
      ├─ ui.js          ルーター・共通UI（表・モーダル・トースト）
      ├─ sync.js        クラウド同期クライアント
      ├─ audit.js       訂正・削除履歴（監査ログ）
      ├─ payrolltax.js  源泉徴収（電算特例）・年末調整の計算
      ├─ app.js         ブートストラップ・ナビ・期間状態
      └─ pages/         各画面（仕訳/経費/取込/請求/固定資産/給与/部門/証憑/決算/申告/電子申告/履歴 ほか）
server/                 クラウド同期サーバー（依存ゼロ・Nodeのみ）
├─ server.js
└─ README.md
```

## 動作確認

会計計算ロジック（試算表・BS/PL・消費税・元帳残高）と、ブラウザ実機での
仕訳入力・請求書作成・PDF表示・売上計上フローを検証済みです。

## ご注意

- 本ソフトは実務のたたき台（MVP）です。消費税の納付額試算は税込経理・原則課税での概算で、
  簡易課税・各種特例・端数処理により実際の申告額と異なる場合があります。
- 会計・税務の最終判断は税理士等の専門家にご確認ください。
