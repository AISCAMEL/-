# クラウド会計（自社版）

freee のような会計ソフトを **自社開発** した最小構成版（MVP）です。
サーバー不要・**ブラウザ単体で動作**し、データはブラウザ内（IndexedDB）に保存されます。

## 使い方

`accounting/index.html` をブラウザで開くだけです。ビルドやインストールは不要です。

- ローカルで開く場合：`index.html` をダブルクリック
- サーバーに置く場合：`accounting/` フォルダをそのままアップロード（静的ホスティングでOK）

> データはお使いのブラウザ内にのみ保存されます。別のPC・ブラウザとは共有されません。
> 共有・引っ越し・バックアップは、**設定 → バックアップ**の書き出し/取り込み（JSON）で行います。

## 主な機能

| 区分 | 機能 |
|------|------|
| 取引 | **仕訳帳**（複式簿記・貸借自動チェック）、**経費・入出金**（かんたん入力→仕訳自動生成）、**請求書 / 見積書**（PDF・印刷・売上計上・入金記録）、**取引先**マスタ |
| レポート | **総勘定元帳**、**合計残高試算表**、**決算書（貸借対照表 BS / 損益計算書 PL）**、**消費税集計**（インボイス対応・税率別・納付税額の試算） |
| 設定 | 会社情報（請求書表示・適格請求書 登録番号）、勘定科目マスタの追加/編集、バックアップ書き出し/取り込み、初期化 |

### 会計仕様

- **複式簿記**：借方＝貸方の一致を入力時にチェック。5要素（資産・負債・純資産・収益・費用）で自動分類。
- **消費税**：税込経理方式。10% / 8%（軽減税率）の税区分に対応。税率区分ごとに端数処理。
- **インボイス**：会社の適格請求書発行事業者 登録番号（T＋13桁）を請求書に表示。税率別の対象額・消費税を明記。
- **会計年度**：期首月を設定可能（初期値 4月始まり）。各レポートは期間で絞り込み可能。

## 画面構成

- ダッシュボード：当期の売上・費用・純利益・現預金残高、最近の仕訳、未入金請求書
- 仕訳帳 / 経費・入出金 / 請求書 / 見積書 / 取引先
- 総勘定元帳 / 試算表 / 決算書(BS・PL) / 消費税集計
- 設定

## 技術構成

- 純粋な HTML / CSS / JavaScript（ビルド不要・外部依存なし・ES モジュール不使用）
- 永続化：IndexedDB（`assets/js/db.js`）
- 画面遷移：ハッシュルーター（`assets/js/ui.js`）

```
accounting/
├─ index.html
└─ assets/
   ├─ css/app.css
   └─ js/
      ├─ util.js        金額・日付・DOM ユーティリティ
      ├─ accounts.js    勘定科目マスタ・税区分の定義
      ├─ db.js          IndexedDB・初期データ・バックアップ
      ├─ store.js       ドメイン層（設定/科目/取引先/仕訳/請求書）
      ├─ reports.js     帳票計算（元帳・試算表・BS/PL・消費税）
      ├─ ui.js          ルーター・共通UI（表・モーダル・トースト）
      ├─ app.js         ブートストラップ・ナビ・期間状態
      └─ pages/         各画面
```

## 動作確認

会計計算ロジック（試算表・BS/PL・消費税・元帳残高）と、ブラウザ実機での
仕訳入力・請求書作成・PDF表示・売上計上フローを検証済みです。

## ご注意

- 本ソフトは実務のたたき台（MVP）です。消費税の納付額試算は税込経理・原則課税での概算で、
  簡易課税・各種特例・端数処理により実際の申告額と異なる場合があります。
- 会計・税務の最終判断は税理士等の専門家にご確認ください。
